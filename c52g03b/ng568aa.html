 clearScope()
 Clear the scope setting for the workarea
------------------------------------------------------------------------------
 Prototype

     ERRCODE clearScope(
                         WORKAREAP wa
                       )

 Arguments

     wa is a pointer to self.

 Description

     Clears the scope information currently in use for a specified workarea.

     Note:  This method is reserved for future implementation.

 Files:  Header file is Rdd.api.
See Also: DBSCOPEINFO WORKAREA setScope()
