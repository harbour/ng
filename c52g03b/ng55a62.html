 orderListFocus()
 Select the controlling order
------------------------------------------------------------------------------
 Prototype

     ERRCODE orderListFocus(
                             WORKAREAP wa,
                             USHORT order
                           )

 Arguments

     wa is a pointer to self.

     order is a numeric value that indicates the Order number to select.

 Description

     Selects the logical Order of the table based on order.

 Default Behavior

     You must implement the default behavior of this method through a
     subclass.

 Implementation Notes

     .  Before you can attach an Order to the workarea, write the
        workarea's buffer to disk through a SELFGOCOLD() call.

     .  The number in order is the position of the Order in the
        controlling key-list.

 Files:  Header file is Rdd.api.
