 setRel()
 Set relation into parent file
------------------------------------------------------------------------------
 Prototype

     ERRCODE setRel(
                     WORKAREAP wa,
                     DBRELINFOP dbRelInfo
                   )

 Arguments

     wa is a pointer to self.

     dbRelInfo is a pointer to a structure containing information on a
     relation.

 Description

     Adds dbRelInfo to the wa->relations list.

 Default Behavior

     Sets a relation--as defined by dbRelInfo--into the parent file wa.  A
     copy of the dbRelInfo structure is placed in the wa->relations list.

     setRel() notifies the child of its additional parent by calling
     childStart().

     Warning!  The dbRelInfo->parent pointer must be the same as the
     workarea pointer that is passed as the first argument.

 Implementation Notes

     .  If you replace this method in your RDD, locate the end of the
        wa->relations chain and mark the next element to point to your new
        relation.

 Files:  Header file is Rdd.api.
