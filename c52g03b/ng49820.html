 delete()
 Mark a row (record) for deletion
------------------------------------------------------------------------------
 Prototype

     ERRCODE delete(
                     WORKAREAP wa
                   )

 Arguments

     wa is a pointer to self.

 Description

     Marks the row at the current cursor location in wa for deletion.

 Default Behavior

     You must implement the default behavior of this method through a
     subclass.

 Implementation Notes

     .  The delete() method marks the data item at the workarea's
        cursor for deletion.  The delete() mechanism for databases must deal
        with the current workarea cursor after the deletion.

     .  If your RDD supports relations, update them with a forceRel()
        prior to attempting the deletion.

     .  On workareas marked as wa->readonly, raise a recoverable
        error.  On workareas marked as wa->shared, you must address network
        contingencies.

     .  Some RDDs may want to perform a physical table deletion.

 Files:  Header file is Rdd.api.
