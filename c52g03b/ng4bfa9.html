 putRec()
 Replace the current row (record)
------------------------------------------------------------------------------
 Prototype

     ERRCODE putRec(
                     WORKAREAP wa,
                     BYTEP buffer
                   )

 Arguments

     wa is a pointer to self.

     buffer is the buffer at the workarea cursor.

 Description

     putRec() replaces the current row's data buffer with the contents of
     buffer.  This method is most notably used to transfer data between two
     similar workareas.

 Default Behavior

     You must implement the default behavior of this method through a
     subclass.

 Implementation Notes

     .  It is assumed that buffer points to the buffer at the workarea
        cursor.  putRec() should call goHot() prior to flushing the buffer
        contents to disk.

 Files:  Header file is Rdd.api.
