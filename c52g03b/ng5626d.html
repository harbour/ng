 clearFilter()
 Clear the active filter expression
------------------------------------------------------------------------------
 Prototype

     ERRCODE clearFilter(
                          WORKAREAP wa
                        )

 Arguments

     wa is a pointer to self.

 Description

     Clears the filter expression currently in use for a specified workarea.

 Default Behavior

     clearFilter() removes the filter expression in wa->filter.  The Items
     held in wa->filter.bExpr and wa->filter.cExpr are both released and set
     to NULL.

 Implementation Notes

     .  Verify that all Items are properly released upon any
        reimplementation of clearFilter().

 Files:  Header file is Rdd.api.
