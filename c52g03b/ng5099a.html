 sysName()
 Obtain the name of the replaceable database driver (RDD) subsystem
------------------------------------------------------------------------------
 Prototype

     ERRCODE sysName(
                      WORKAREAP wa,
                      BYTEP name
                    )

 Arguments

     wa is a pointer to self.

     name is a pointer to a buffer that is assigned the name to the RDD.

 Description

     Obtains the name of the driver subsystem.

 Default Behavior

     You must implement the default behavior of this method through a
     subclass.

     Warning!  name must point to a preallocated buffer of at least 16
     bytes.

 Implementation Notes

     .  Your RDD should implement sysName() by simply returning the
        name of your driver.  For example, the DBFNTX driver copies "DBFNTX"
        into name and returns.  There is no need to maintain the inheritance
        chain for sysName().

 Files:  Header file is Rdd.api.
