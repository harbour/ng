 zap()
 Physically remove all rows (records) from the data store (table)
------------------------------------------------------------------------------
 Prototype

     ERRCODE zap(
                  WORKAREAP wa
                )

 Arguments

     wa is a pointer to self.

 Description

     Physically removes all rows from the data store referenced by wa.

 Default Behavior

     You must implement the default behavior of this method through a
     subclass.

 Implementation Notes

     .  You may only implement zap() with the appropriate network
        locks in place while the workarea is not in read-only mode.

 Files:  Header file is Rdd.api.
