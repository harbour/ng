 The Runtime Environment
------------------------------------------------------------------------------

     Using ANSITERM.LIB requires that ANSI.SYS be installed on the user's
     computer.  To accomplish this, include the following statement in the
     user's CONFIG.SYS:

     DEVICE=ANSI.SYS
