 Installing DBFNTX Driver Files
------------------------------------------------------------------------------

     DBFNTX is supplied as the file DBFNTX.LIB.

     The CA-Clipper installation program installs this driver as the default
     in the \CLIPPER5\LIB subdirectory on the drive that you specify, so you
     need not install the driver manually.

     Important!  Before installing CA-Clipper, you may want to rename the
     DBFNTX.LIB that currently resides in your \CLIPPER5\LIB directory to
     DBFNTX.001. The new version, when installed, will overwrite DBFNTX.LIB.
     If you do not rename or otherwise protect the old version of DBFNTX.LIB,
     you will lose it.
