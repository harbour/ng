 Chapter 8: DBPX Driver Installation and Usage
------------------------------------------------------------------------------

     DBPX is the Paradox 3.5 compatible RDD for CA-Clipper.  It connects to
     the low-level database management subsystem in the CA-Clipper
     architecture.  When you use the DBPX RDD, you add a number or features,
     including the ability to:

     .  Create access to and modify Paradox tables, records, and
        fields

     .  Create, select, and activate secondary indexes on Paradox
        tables

     .  Create and modify Paradox table structures, including primary
        index fields

     .  Use explicit record and file locks with concurrent execution
        of other CA-Clipper applications

     .  Import Paradox tables directly into CA-Clipper  arrays
