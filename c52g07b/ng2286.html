 Chapter 9: Alternate Terminal Drivers
------------------------------------------------------------------------------

     CA-Clipper provides several Alternate Terminal Drivers to allow your
     applications to run in a wider variety of environments.  This chapter
     discusses how Alternate Terminal Drivers fit into the overall CA-Clipper
     architecture, as well as how to install and use each of the supplied
     terminal drivers:  ANSITERM, NOVTERM, and PCBIOS.
