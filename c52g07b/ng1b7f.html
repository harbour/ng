 Chapter 5: DBFMDX Driver Installation and Usage
------------------------------------------------------------------------------

     The DBFMDX database driver provides access to dBASE IV (.dbf), (.mdx),
     and (.dbt) file formats.  The driver also supports dBASE IV-compatible
     file and record locking schemes, allowing shared access between
     CA-Clipper  and dBASE IV programs.  This chapter explains how to install
     DBFMDX and how to use it in your applications.
