  SIX???/1020   Data Type Error

  Problem:   The value assigned to a FIELD variable was of the wrong type.

  Solution:  Correct the program.  If assigning a FIELD variable from
             some other variable, make sure the other variable has been
             initialized (i.e., is not NIL).
