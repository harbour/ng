  Why am I getting four Unresolved External errors (__ZCMP, etc...)
  when I link?:

  If you are using the International SIx 3.0 release, you must link in one
  of the included nation modules to select the index sorting table for the
  language you are using.

  For example, to use the American/English sorting order, you'd use the
  SIXUK.OBJ file:

    BLINKER FI test, sixuk @six3

  These .OBJ files can be found in the directory where all of your other
  SIx Driver .OBJ files were installed.  Unless you specified a different
  directory during installation, this would be \SIX3\OBJ.


    NOTE:  These foreign nation modules are only included, and required,
           with the International SIx 3.0 release.  See the "Notes For
           International Users ONLY" section earlier in this documentation
           for more details on how to use these foreign nation modules.
