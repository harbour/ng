  What commands can I optimize?:

  Mach SIx includes built-in support for the following Clipper commands:

          AVERAGE
          CONTINUE
          COUNT
          COPY [SDF] [DELIMITED]
          DELETE
          DISPLAY
          INDEX ON
          LABEL
          LIST
          LOCATE
          RECALL
          REPLACE
          REPORT
          SET FILTER
          SORT
          SUM
          TOTAL


  Mach SIx will optimize any of the above commands, that contain one or more
  active index key expressions in the FOR condition.  Note that Mach SIx
  will only optimize commands with a FOR condition and the ALL scope ( the
  ALL scope is the default for all supported commands).   Specifying a WHILE
  condition or alternate scope will cancel the benefits of optimization.
