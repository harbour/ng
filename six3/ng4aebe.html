  PACK:

  This command behaves similarly to the standard Clipper PACK command.
  However, when you issue a PACK command with the SIx Driver active, not
  only will your database file be purged of deleted records, but any assoc-
  iated memo files (.FPT, .SMT, or .DBT formats) will be purged of free
  blocks as well.  This is done with an internal call to the Sx_MemoPack()
  function.
See Also: MEMOPACK Sx_MemoPack()
