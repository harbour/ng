  Sx_FileOrder():

  Syntax:  Sx_FileOrder()

  Returns:  <numeric> Position of current index file in the index file list.

  Description:  This function is provided to duplicate the behaviour of
                the IndexOrd() function in CA-Clipper releases prior to 5.2.

          NOTE:  This is a compatibility function and is provided only for
                 backwards SIx Driver compatiblity.  The source code is
                 included in OBSOLETE.PRG.
