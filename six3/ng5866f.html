  SIX???/1023   Exclusive Required

  Problem:   The operation being attempted requires exclusive use of the
            database (.DBF) file but the work area was opened for shared
            access.

  Solution:  Correct the program.  The PACK, REINDEX and ZAP commands
             require exclusive access to the database (.dbf) file.  To obtain
             exclusive access, use the EXCLUSIVE clause of the USE command.
