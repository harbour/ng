  SIX???/634  Not SIx Driver

  Problem:   A SIx Driver-only function was attempted with a database file
             (.DBF) that was opened with a different RDD (usually DBFNTX).

  Solution:  Change program to use the SIx Driver to open the desired
             database file (.DBF).
