  Sx_SetTagOrder():


  Syntax:  Sx_SetTagOrder([cTag | nOrder] [,cIndexName])

            cTag       = Name of the tag to make active
            nOrder     = Position of order in the order list
            cIndexName = Name of the file that contains the tag or order

  Returns:   <logical> .T. if order was successfully changed, .F. if not.

  Description:  This function works exactly like Sx_SetTag().

          NOTE:  Sx_SetTagOrder() is an obsolete function. It is
                 provided for compatibility only.  See OBSOLETE.PRG
                 and OBSOLETE.CH.


    NOTE:  This function is NOT supported under SIXNTX.
