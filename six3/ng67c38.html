  Max FPT/SMT memo file size of 4Gb:

  The theoretic FPT/SMT file size limitation is 4Gb.  However, limits set by
  the current releases of DOS and OS/2 limit file sizes to 2Gb.  Also,
  DOS 3.2 and earlier only support a maximum file size of 32 Mb.

  The maximum FPT/SMT file size does NOT depend on the memo block size in
  use.

  The memo block size is set with the Sx_SetMemoBlock() function.
See Also: Sx_SetMemoBlock()
