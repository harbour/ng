  SIXVER.EXE:

  The SIXVER.EXE utility will scan your SET LIB environment path, searching
  for the first SIX3.LIB file it comes to (the same one your linker would
  use).  SIXVER.EXE then displays the following information about it:

    . Location of the SIX3.LIB file (Complete path)
    . SIX3.LIB Date Stamp
    . SIX3.LIB Time Stamp
    . SIX3.LIB File Size
    . Complete version number, including sub-version (i.e., 3.00.00)

  This utility takes no command-line parameters.
