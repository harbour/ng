  Sx_IsDBT():

  Syntax:  Sx_IsDBT()

  Returns:  .T. if DBT.OBJ was linked in, .F. if not.

  Description:  Indicates whether or not the DBT.OBJ was linked into the
                application.  This allows the developer to determine, before
                the data file is opened, which type of memo file, if any,
                should exist (.FPT, .SMT or .DBT).

  Example:

    ? Sx_IsDBT()  // If this returns .T., you can only use DBT memos
See Also: Sx_MemoExt() RDD_Info()
