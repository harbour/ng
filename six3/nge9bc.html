  m6_FiltCopy():

  Purpose:  Creates a new instance of the specified filter.

   Syntax:  m6_FiltCopy( <nHandle> )

    Where:  <nHandle>     --  Is any valid handle to a filter.

  Returns:  A handle to the copy of the filter.

     Note:  *Always free RYO filters that are no longer being used.*

  Example:

         #include "SIXCDX.CH"
         #include "MACHSIX.CH"

         USE Customer via "SIXCDX"

         SET INDEX TO State, Age

         //........................................create two filters
         nHandle1 := m6_NewFilter( "State='CA'" )
         nHandle2 := m6_NewFilter( "Age=25" )

         //.......save filter1 before it gets stomped on by the join
         nHandle3 := m6_FiltCopy( nHandle1 )

         //................................join filter2 into filter1
         m6_FiltJoin( nHandle1, nHandle2, JOIN_UNION )
