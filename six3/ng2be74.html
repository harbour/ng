  Sx_IsReadOnly():

  Syntax:   Sx_IsReadOnly()

  Returns:  Logical .T. if current or aliased database was opened in
            READONLY mode, or .F. if not.

  Description: See Return value above.

  Example:

          USE test VIA "SIXCDX" READONLY   // Open TEST.DBF in READONLY mode
          ? Sx_IsReadOnly()                // .T.
          USE test VIA "SIXCDX"            // Open TEST.DBF in default mode
          ? Sx_IsReadOnly()                // .F.
See Also: Sx_IsFLocked() Sx_IsShared()
