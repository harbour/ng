  SIX???/1010   Read Error

  Problem:   A read error occurred on the specified file.

  Solution:  If the file is on a floppy disk, make sure the disk is
             properly seated in the drive.  In a network environment, make
             sure the network connection is still valid; check for problems
             at the server.

        NOTE:  Another common cause of the 1010 Read Error is when your
               application has insufficient STACK space.  This problem
               can also result in Internal Error 650.  You may need to
               add or increase the STACK or PROCEDURE DEPTH settings
               in your link script.  (This STACK is not the same as
               the one in your CONFIG.SYS file.

               For example, try adding one (but not both) of the follow-
               ing lines to your link script (using Blinker for this
               example:

                  STACK 10240

                  BLINKER PROCEDURE DEPTH 100

              After relinking, delete and rebuild all your index files.
