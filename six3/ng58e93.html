  SIX???/601  Seek Error

  Problem:   A seek error (file positioning) occurred on the specified
             file.

  Solution:  If the file is on a floppy disk, make sure the disk is
             properly seated in the drive.  In a network environment, make
             sure the network connection is still valid; check for problems
             at the server.
