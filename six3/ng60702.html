  Using the SIx Driver With Other RDD's:

  When using the SIx Driver with other RDD's, always be sure to place the SIx
  Driver library file before other RDD libraries in your link command or
  link script, unless specifically instructed differently.

  Also, be sure to include the SIx Driver header file in your source file(s).
