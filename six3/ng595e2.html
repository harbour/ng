  SIX???/610  Bad Index Expression

  Problem:   An invalid expression was encountered while creating or
             opening an index.

  Solution:  Make sure the key expression (and condition expression, if
             applicable) is a valid Clipper expression.  If a User-Defined
             Function (UDF) is used in an expression make sure it is
             available in all applications that use the index.

             Another common cause of the 610 error is when you are attempting
             to open a FoxBASE-format, non-compact .IDX index file.  SIx 3.0
             only supports the FoxPro-format, compact .IDX and compound .CDX
             index formats.
