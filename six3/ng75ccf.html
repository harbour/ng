------------------------------# OBJ Files #-----------------------------------
 DBT.OBJ        Supports use of Clipper .DBT files with SIXCDX and/or SIXNSX
 FPT.OBJ        Supports use of FoxPro .FPT files with SIXNSX and/or SIXNTX
 SMT.OBJ        Supports use of SIXNSX .SMT files with SIXCDX and/or SIXNTX
 NOMEMO.OBJ     Eliminates default memo support from SIx Driver workareas
 SIXCDX.OBJ     Makes SIXCDX the default RDD; Doesn't link in DBFNTX
 SIXMDX.OBJ     Makes SIXMDX the default RDD; Doesn't link in DBFNTX
 SIXNSX.OBJ     Makes SIXNSX the default RDD; Doesn't link in DBFNTX
 SIXNTX.OBJ     Makes SIXNTX the default RDD; Doesn't link in DBFNTX
------------------------------# Utilities #-----------------------------------
 BLIGPF.EXE     Extracts GPF Error Information From Blinker 3.x MAP Files
 CDXINFO.EXE    Displays .CDX/.IDX tag information
 DBCRE8.EXE     Creates test databases with pseudo-random data
 DBFINFO.EXE    Displays data table information; Fixes corrupt file headers
 EXOGPF.EXE     Extracts GPF Error Information From CA-ExoSpace MAP Files
 NSXINFO.EXE    Displays .NSX tag information
 SEEK.EXE       Locates a specified file in a specific environment path
 SIXINI.EXE     Encrypts/Decrypts Data Dictionary (.INI) files
 SIXVER.EXE     Reports SIx 3.0 Library Revision Numbers
 STRTRAN.EXE    Performs binary search and replaces within specified files
