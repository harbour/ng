  Sx_IsShared():

  Syntax:   Sx_IsShared()

  Returns:  Logical .T. if current or aliased database was opened in
            SHARED mode, or .F. if not.

  Description: See Return value above.

  Example:

          USE test VIA "SIXCDX" SHARED     // Open TEST.DBF in SHARED mode
          ? Sx_IsShared()                  // .T.
          USE test VIA "SIXCDX" EXCLUSIVE  // Open TEST.DBF in EXCLUSIVE mode
          ? Sx_IsShared()                  // .F.
See Also: Sx_IsFLocked() Sx_IsReadOnly()
