  Sx_SetFileOrd():

  Syntax:  Sx_SetFileOrd( <nFileOrder> )

     <nFileOrder> is the position of an index file in the index file list.


  Returns:  <numeric> The previously set file order position.

  Description:  Selects 1st tag in an index file given the file order
                position in the SET INDEX TO list.  Works like dbSetOrder()
                did under Clipper 5.01a.

         NOTE:  This function is obsolete.  It is provided for compatability
                only.  Clipper source code can be found in OBSOLETE.PRG.
