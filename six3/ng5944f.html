  SIX???/605  Insufficient Memory

  Problem:   There is insufficient conventional memory for the requested
             operation.

  Solution:  For safety, quit the application.  Make more memory available
             before running the application again.  Remove unneeded resident
             utilities or device drivers.  Opening fewer database or index
             files may make more memory available.
