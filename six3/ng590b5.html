  SIX???/603  Open Error (semaphore file)

  Problem:   The specified semaphore file (.SEM) could not be opened.

  Solution:  Check to make sure that the specified file exists.  If the
             file is not in the directory where the application runs, use SET
             DEFAULT or SET PATH to make the file accessible or specify the
             full path name where the file can be found.  In a network
             environment, make sure the application has the necessary rights
             to access the file.
