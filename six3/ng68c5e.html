  Can perform a PACK on FPT/SMT/DBT memo files:

  When you issue a PACK command with the SIXCDX or SIXNSX driver active,
  not only will your database file be purged of deleted records, but any
  associated memo files (FPT, SMT, or DBT) will be purged of free blocks
  as well.  This is done with an internal call to the Sx_MemoPack() function.

  If you want to pack a memo file itself, without packing the database file,
  you can do so by simply calling the MEMOPACK command or the Sx_MemoPack()
  function.

  Good-Bye Memo Bloat!
