  Sx_IsFLocked():

  Syntax:   Sx_IsFLocked()

  Returns:  Logical .T. if SHARED database file is currently locked by
            FLock(), otherwise it returns .F.

  Description: See Return value above.

  Example:

          USE test VIA "SIXCDX" SHARED     // Open TEST.DBF in SHARED mode
          ? Sx_IsFLocked()                 // .F.
          FLock()                          // Lock TEST.DBF
          ? Sx_IsFLocked()                 // .T.
See Also: Sx_IsShared() Sx_IsReadOnly() Sx_IsLocked()
