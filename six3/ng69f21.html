  RDD Utilities:

  The SIx Driver's RDD functions give you control over Clipper's Replaceable
  Database Driver system.  You can get information about all RDD's loaded,
  such as number of RDD's available, RDD names, whether the RDD has been
  loaded or not, default file extensions, etc.  You can determine which
  RDD is currently active, as well as change the default RDD.
