  Record Edit Rollbacks:

  Single record rollbacks allow you to edit field values directly without
  worrying about getting the original values back.  Any time before the
  record is written to disk you can use the SIx Driver's Sx_RollBack()
  function to restore the original record.  Sx_RollBack() can also restore
  the current record for all child tables.

  NOTE:  See ROLLBACK.PRG for an example of Sx_RollBack() in action.
