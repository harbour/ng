  m6_RefreshFilter():

  Purpose:  Used to refresh Mach SIx's internal filter record list for
            updates to shared files.

   Syntax:  m6_RefreshFilter()

  Returns:  NIL.

  Example:

         #include "SIXCDX.CH"
         #include "MACHSIX.CH"

         USE Names SHARED INDEX Name, Salary, State, Date

         //............................................set filter condition
         SET FILTER TO State = "CA"

         //.............................................activate the filter
         GO TOP

         DO WHILE LastKey() != 27

           //..................................browse the filtered records
           Browse()

           //............................................refresh the filter
           m6_RefreshFilter()

         ENDDO
See Also: REFRESH FILTER SET FILTER
