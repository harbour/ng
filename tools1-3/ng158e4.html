 GETLINES()
 Determines the number of lines after which the screen display pauses
------------------------------------------------------------------------------
 Syntax

     GETLINES() --> nLineNumber

 Returns

     GETLINES() returns the value established with SETLINES()

 Description

     GETLINES() queries the value previously set with SETLINES().

 Examples

     .  Set a value:

        SETLINES(20)

           LIST ...

     .  Query that value:

        ? GETLINES()               // Result: 20
