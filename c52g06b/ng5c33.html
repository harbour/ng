 Valedit.prg
 Example of using Get objects
--------------------------------------------------------------------------------

     ValEdit( <nRow>, <nCol>, <xValue>,
        [<cPicture>], [<cColor>] ) --> value

     Uses a Get object to edit value <xValue> at screen location given by
     <nRow>, <nCol>.  Uses picture <cPicture>, color <cColor> if passed.
