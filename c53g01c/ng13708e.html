     GetNew([<nRow>], [<nCol>], [<bBlock>], [<cVarName>],
        [<cPicture>], [<cColorSpec>]) --> oGet

 Returns

     GetNew() returns a new Get object with the row, col, block, picture, and
     colorSpec instance variables set from the supplied arguments.
