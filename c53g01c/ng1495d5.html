 PushButton
 Create a push button
------------------------------------------------------------------------------
 Description

     Places a push button at the indicated position on the screen.

 Examples

     .  This example creates an object which is placed at row 2 and
        column 2 and has a caption of "Exit":

        oBtn:PushButton(2,2,"Exit")
