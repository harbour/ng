 C2072   Ambiguous match pattern in #translate/#command

     Explanation:  You specified a #translate/#xtranslate or
     #command/#xcommand directive containing two consecutive optional
     match-marker clauses with no intervening keyword.

     Action:  This is not allowed since the preprocessor has no way of
     knowing which match marker to use for matching input text.  Refer to the
     Language Reference chapter of the Reference guide for more information
     and examples using the translation directives.
