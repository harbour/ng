 R1003   Ignoring text: '<text>'

     Explanation:  A statement in a make file contains extraneous
     characters.  The characters are ignored.

     Action:  Examine the make file to ensure that the extraneous text is
     not the result of some other error in the file.  If not, delete the
     characters from the make file.
