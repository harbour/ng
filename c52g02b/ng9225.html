 C2056   Unexpected end of file in #define

     Explanation:  You specified a #define directive containing an
     end-of-file CHR(26) mark.  The syntax for this and all other
     preprocessor directives can be found in the Language Reference chapter
     of the Reference guide.

     Action:  Correct and recompile.
