 ert0061 Input/output error on output file <filename>

     Explanation:  The linker cannot read from and/or write to the named
     output file.

     Action:  Since this error message normally indicates a corrupted
     file or disk, try running the DOS CHKDSK /F utility to see if your disk
     is corrupted and correct the problem.
