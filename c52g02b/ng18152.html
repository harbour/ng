 eut0035 Write to expanded (EMS) or extended (XMS) memory failed

     Explanation:  An attempt by the linker to write to expanded/
     extended memory failed.  This is usually indicative of either a
     failure of the memory or a corruption of the memory by some other
     program.

     Action:  Temporarily disable the EMS/XMS memory and re-link.
