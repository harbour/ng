 C3039   Phase error

     Explanation:  An unknown symbol has been encountered in the
     compiler's second pass.

     Action:  Usually occurs when .PRGs are too big or when the .PRG
     contains complicated and long control structures such as nested CASE
     statements.  Try simplifying the program.

     Can also occur if the program contains two OTHERWISE statements in a DO
     CASE construct.
