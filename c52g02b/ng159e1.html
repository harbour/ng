 ert0077 Cannot find startup code 'RTLINKST.COM'

     Explanation:  When generating a .PLL with the PRELINK option,
     .RTLink could not locate RTLINKST.COM.

     Action:  RTLINKST.COM contains the necessary startup code that must
     be written to the .PLL.  When this file is needed, only the directory in
     which RTLINK.EXE was originally found is searched--no other path
     settings are used.
