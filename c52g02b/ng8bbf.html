 C2052   MEMVAR declaration follows executable statement 

     Explanation:  You specified a MEMVAR statement following an
     executable statement.  All variable declaration statements must precede
     any executable statements in a procedure or function definition.

     Action:  Move the MEMVAR statement before any executable statements
     within the procedure or function definition and recompile.
