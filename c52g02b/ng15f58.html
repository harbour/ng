 ert0085 Freeformat parser tables too big to fit in memory

     Explanation:  There is not enough memory on your machine for .RTLink
     to load the input parser tables from RTLINK.DAT.

     Action:  In order to run .RTLink, you will need to increase the
     amount of free memory on your machine.
