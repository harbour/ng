 ert0033 Invalid format in transfer (.PLT) file '<filename>' (possibly caused
         by a bad .PLL/.PLT name)

     Explanation:  Either the .PLT file is not valid or it has somehow
     been corrupted.

     Action:  Check the name of the file first.  If it is incorrect,
     correct the name specified with the PLL option.  If the name of the .PLT
     is correct, regenerate the file by regenerating its associated .PLL.
