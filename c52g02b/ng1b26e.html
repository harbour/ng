 BASE/1002    Undefined alias

     Explanation:  You specified an alias not currently associated with
     any work area.

     Action:  Make sure the appropriate database file is open. Check to
     see if more than one work area uses the same alias at the same time.  If
     so, designate a unique alias for each work area.  The SELECT() function
     can be used to see if an alias is currently associated with any work
     area.

     See Also:  Basic Concepts chapter in the Programming and Utilities
     guide
