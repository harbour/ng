 wrt0009 Unrecognized map type '<char>' .... ignored

     Explanation:  One of the map specifications entered just after the
     MAP option is not valid.

     Action:  The only valid map specifications are the letters A, N, and
     S, which can be specified in either upper or lowercase.  Explanations
     can be found in The CA-Clipper Linker--RTLINK.EXE chapter of the
     Programming and Utilities guide.
