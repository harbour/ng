 ert0097 Invalid MAP flag <type>

     Explanation:  You have specified the MAP option with an invalid
     report option.

     Action:  The valid MAP options are S, N, and A.  Explanations can be
     found in The CA-Clipper Linker--RTLINK.EXE chapter in the Programming
     and Utilities guide.
