 C3040   Unexpected end of intermediate file

     Explanation:  An unexpected EOF was encountered while reading from
     the intermediate (temporary) file.

     Action:  This error occurs because the file was not fully written to
     disk due to one the following:

     1.  Insufficient diskspace.

     2.  Insufficient write privileges on a network.

     3.  Not enough file handles.
