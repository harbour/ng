 ert0049 Cannot create map file '<map>'

     Explanation:  The linker cannot create the specified map file.
     Normally this is caused by an invalid map filename, a protection on the
     map file, or a nonexistent directory as part of the map name.

     Action:  If you get this message check to ensure that the indicated
     map name is valid.
