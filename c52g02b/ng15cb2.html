 ert0083 Cannot find RTLINK.DAT (freeformat parser data)

     Explanation:  .RTLink was unable to locate the RTLINK.DAT file.

     Action:  RTLINK.DAT is needed each time you execute RTLINK.EXE.
     Only the directory in which RTLINK.EXE was originally found is searched
     for this file--no other path settings are used.
