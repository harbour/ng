 BASE/1234    String overflow: REPLICATE

     Explanation:  The arguments to REPLICATE() specified the creation of
     a character value whose length exceeded the maximum length.

     Action:  Correct the program.  The maximum length of a character
     value in CA-Clipper is 65519 characters.
