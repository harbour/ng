 C3030   Write error to OBJ

     Explanation:  The compiler created an object file but cannot write
     to the file.  This error can occur if you run out of disk space during
     the compile.

     Action:

     1. Free some disk space and recompile.

     2. Direct the compiler to write the object file (.OBJ) to another drive
        using the /O option.
