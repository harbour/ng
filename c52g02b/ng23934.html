 DBFNTX/1010  Read error

     Explanation:  A read error occurred on the specified file.

     Action:  If the file is on a floppy disk, make sure the disk is
     properly seated in the drive.  In a network environment, make sure the
     network connection is still valid; check for problems at the server.

     See Also:  Network Programming chapter in the Programming and
     Utilities guide
