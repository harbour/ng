 167 NEW: NIL
--------------------------------------------------------------------------------

     NIL has been added to Clipper 5.0 as a new data type.  This value
     is automatically assigned to all uninitialized variables (except
     for publics).  Also, when parameters are omitted in a procedure or
     function call, NIL is automatically passed in their place.
