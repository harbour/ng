 67  NEW: Get:reader
--------------------------------------------------------------------------------

     Get:reader is a new exported instance variable that can be used to
     implement special read behaviors for any Get.  Get:reader allows
     particular Get objects to have specialized read behaviors without
     changing the standard READMODAL() function.  This preserves
     compatibility for Gets which are to be handled in the customary way
     and also eliminates potential conflicts between different
     extensions to the GET/READ system.
