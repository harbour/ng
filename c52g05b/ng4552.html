 34  NEW: FIELDBLOCK()
--------------------------------------------------------------------------------

     FIELDBLOCK() is a new function that builds a code block designed to
     assign or retrieve a designated database field.  When executed with
     an argument, the code block assigns the value of the argument to
     the field.  When executed without an argument, the code block
     retrieves the current field value.
