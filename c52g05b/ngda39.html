 148 NEW: ->
--------------------------------------------------------------------------------

     The alias operator (->) has been enhanced in several ways:

     1. The alias operator (->) now supports the FIELD and MEMVAR alias
        names to refer explicitly to a database field or memory
        variable.

     2. In addition, database fields may be aliased using their work
        area number by enclosing the number in parentheses before the
        alias operator (e.g., (5)->CustName).
