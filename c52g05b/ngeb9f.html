 160 NEW: _xgrab()
--------------------------------------------------------------------------------

     _xgrab() is a new function in the Extend System that allows a C or
     Assembly language function to allocate memory from Clipper's fixed
     heap.  A runtime error is generated if the allocation is
     unsuccessful.
