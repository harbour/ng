 47  CHANGED: GETENV()
--------------------------------------------------------------------------------

     The syntax of the GETE() function has been changed to GETENV().
     The shortened version, however, is still syntactically correct.
     This function, originally defined in EXTEND.LIB, is now part of
     CLIPPER.LIB.
