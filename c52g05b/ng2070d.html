 137 NEW: Options Menu
--------------------------------------------------------------------------------

     The Options menu is new.  It contains the Preprocessed Code, Line
     Numbers, and Exchange Screens commands that were on the View menu
     along with many new commands in Clipper 5.00.  The new commands are
     as follows:

     Table: New Options Menu Commands
     -----------------------------------------------------------------
     Menu Command       Action
     -----------------------------------------------------------------
     Swap on Input      Controls exchanging screens when user input is
                        requested
     Codeblock Trace    Toggles trace back to code block definitions
                        when code blocks are evaluated
     Menu Bar           Toggles the display of the menu bar
     Mono Display       Toggles monochrome display
     Colors             Invokes the Set Colors Window
     Tab Width          Sets the display width of Tabs in the Code
                        Window
     Save Settings      Save the current configuration of the Debugger
                        to a script file
     Restore Settings   Restore a saved configuration making it the
                        current state of the Debugger
     -----------------------------------------------------------------
