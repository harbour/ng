 118 CHANGED: SET KEY
--------------------------------------------------------------------------------

     1. SETCANCEL():  If SETCANCEL() is on (.T.), SET KEY 302 is
        ignored and pressing Alt-C always terminates the application.

     2. Alt-D:  If the Debugger is enabled (e.g., with ALTD(1)), SET
        KEY 288 is ignored and pressing Alt-D always invokes the
        Debugger.  In both cases, however, disabling the key (with
        SETCANCEL(.F.) or ALTD(0)) allows you to define and use a SET
        KEY equivalent.
