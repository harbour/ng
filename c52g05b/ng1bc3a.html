 76  CHANGED: TBColumn:width
--------------------------------------------------------------------------------

     If this instance variable is not explicitly set, the width of the
     column will be the greater of:

     1.  Length of heading
     2.  Length of footing
     3.  Length of data at first evaluation of data block

     The width of the displayed data will be the length at first
     evaluation of the data block.  If this instance variable is
     explicitly set, the width of the column will be TBColumn:width.
     Displayed headings, footings, and data will all be truncated to
     this width when necessary.

     The width of the displayed data will be length at first evaluation
     of data block for all data types other than character.  Character
     data will be extended to TBColumn:width for display purposes.
