 CHANGED:  NTX
------------------------------------------------------------------------------

     The NTX driver for CA-Clipper 5.2 enhances the DBFNTX driver supplied
     with earlier versions of CA-Clipper by adding the following features:

     .  Create conditional indexes by specifying a FOR condition.

     .  Create indexes using a record scope or WHILE condition, allowing you
        to INDEX based on the order of another index.

     .  Create both ascending and descending order indexes.

     .  Specify an expression that is evaluated periodically during indexing
        in order to display an index progress indicator.
