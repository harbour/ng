 142 CHANGED: .AND.
--------------------------------------------------------------------------------

     In a departure from Summer '87, Clipper 5.0 shortcuts the
     evaluation of .AND. operands by evaluating the second condition
     only when the first condition is true (.T.).  If the first
     condition is false (.F.), the .AND. operation is false (.F.), and
     the second condition is never evaluated.
