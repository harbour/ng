 CHANGED:  Error.ch
------------------------------------------------------------------------------

     The following generic error codes have been added to Error.ch:

     EG_DUPALIAS:  An attempt has been made to open a database with an alias
     that is already in use.

     EG_BADALIAS:  An attempt has been made to open a database with an alias
     that is not a valid CA-Clipper identifier.

     EG_CYCLICAL:  A subsystem has rejected request to perform a cyclic
     operation. For example, a work area issues a SET RELATION into itself.
