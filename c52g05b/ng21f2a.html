 150 NEW: TEMPPATH, NOIDLE
--------------------------------------------------------------------------------

     The CLIPPER runtime configuration variable has been enhanced.  The
     following lists summarizes the new CLIPPER settings:

     Table: New Clipper 5.01 Configuration Settings
     -------------------------------------------------------------------
     Setting      Meaning
     -------------------------------------------------------------------
     TEMPPATH     Controls the placement of temporary files created
                  during sorting and indexing
     NOIDLE       Prevents Clipper from detecting and taking advantage
                  of idle time during execution of compiled applications
     -------------------------------------------------------------------
