 66  NEW: Get:exitState
--------------------------------------------------------------------------------

     Get:exitState is a new exported instance variable used by the
     Clipper 5.01 READMODAL() system to record the means by which the
     Get object was exited when it was last read.  For a list of the
     exit states, refer to Getexit.ch.
