 79  NEW: EG_NOVARMETHOD, EG_APPENDBLOCK, EG_MEM
--------------------------------------------------------------------------------

     The following new generic error codes have been added:

     Table: New Generic Error Codes
     --------------------------------------------------------------------
     Constant           Meaning
     --------------------------------------------------------------------
     EG_NOVARMETHOD     For errors that occur when attempting to assign
                        an instance variable of an object
     EG_APPENDBLOCK     For when APPEND BLANK is unable to lock a new
                        record at the end of the file
     EG_MEM             For when the amount of conventional memory
                        available may be insufficient to allow continued
                        execution
     --------------------------------------------------------------------
