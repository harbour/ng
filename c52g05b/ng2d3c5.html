 CHANGED:  INDEX command
------------------------------------------------------------------------------

     The INDEX syntax has been modified to include new clauses.  The new
     syntax is:

     INDEX ON <expKey> [TAG <cOrderName>] TO  <xcOrderBagName>
        [FOR <lCondition>] [ALL]
        [WHILE <lCondition>] [NEXT <nNumber>]
        [RECORD <nRecord>] [REST]
        [EVAL <bBlock> [EVERY <nInterval>]
        [UNIQUE] [ASCENDING|DESCENDING]
