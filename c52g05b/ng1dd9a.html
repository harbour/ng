 99  REMOVED: INSPECT WP
--------------------------------------------------------------------------------

     The INSPECT WP command has been deleted.  All Watchpoints and
     Tracepoints are displayed in the Watch Window.  To inspect a
     Watchpoint, use Tab to select the window and Uparrow and Dnarrow to
     move the highlight to the item you want to inspect.  Pressing
     Return displays the selected item in a dialog box where you can
     inspect it.
