 111 REMOVED: F3
--------------------------------------------------------------------------------

     The F3 key, previously used to retype the last command in the
     Command Window, is no longer valid.  Instead, the Command Window
     maintains a command history.  When the window is active, Uparrow
     displays the previously issued commands and Return executes the
     currently displayed command.
