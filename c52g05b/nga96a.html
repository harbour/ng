 117 CHANGED: SET FUNCTION
--------------------------------------------------------------------------------

     The new implementation for SET FUNCTION pre-processes the command
     into SET KEY and KEYBOARD commands causing SET FUNCTION to clear a
     KEY SETting for the same key.  This is incompatible with previous
     releases which maintained separate SET FUNCTION and SET KEY lists.
