 69  NEW: TBrowse:insColumn(<nPos>, <oColumn>)
--------------------------------------------------------------------------------

     TBrowse:insColumn() is a new method that allows a column object to
     be inserted into the middle of a browse.  The return value is a
     reference to the column object being inserted.
