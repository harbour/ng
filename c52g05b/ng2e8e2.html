 NEW:      RDDLIST() function
------------------------------------------------------------------------------

     RDDLIST() is an RDD function that returns a one-dimensional array that
     lists the available RDDs.

     If you do not supply <nRDDType>, all available RDDs, regardless of type,
     are returned.
