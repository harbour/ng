 NEW:      Run menu option
------------------------------------------------------------------------------

     The Next routine (Ctrl-F5) option has been added to the Run menu.  This
     option causes the Debugger to advance to the next routine.  It is
     equivalent to creating a "PROCLINE() == 0" tracepoint.
