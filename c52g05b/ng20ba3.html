 138 NEW: Window Menu
--------------------------------------------------------------------------------

     The Window menu is new.  It contains commands to allow you to
     control the manner in which windows are displayed on the main
     Debugger screen.  The commands are Next, Prev, Move, Size, Zoom,
     Iconize and Tile.
