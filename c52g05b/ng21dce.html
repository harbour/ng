 149 NEW: Set Colors Window
--------------------------------------------------------------------------------

     The Set Colors Window has been added.  This window is opened using
     Options:Colors and closed by pressing Esc.  When the Set Colors
     Window is open, all of the Debugger color settings can be
     inspected.
