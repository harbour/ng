 153 CHANGED: Browse.prg
--------------------------------------------------------------------------------

     Browse.prg is a new source file that contains the source code for
     the standard function, BROWSE().  The source code for this function
     was missing from the Clipper 5.00 system.

     -----------------------------------------------------------------
     . Note

     Note that this file contains the source code for a standard
     function and therefore is not necessarily useful as an example of
     how to create a TBrowse.  If you are looking for sample code for
     TBrowse, Tbdemo.prg is recommended instead.
