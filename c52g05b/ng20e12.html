 140 NEW: Window Operations
--------------------------------------------------------------------------------

     A full range of window operations are available in Clipper 5.01 for
     windows on the main Debugger screen.

     1.  New operations include sizing, moving, tiling, iconizing and
         selecting the previously active window.

     2.  All window operations can be accomplished using the Window menu
         commands.

     3.  Tab and Shift-Tab now select next and previous windows.

     4.  The order for window selection is as follows: Code, Monitor,
         Watch, Callstack, Command.
