 55  CHANGED: INPUT
--------------------------------------------------------------------------------

     The INPUT command has been changed to preserve the previous value
     of the target variable when no new value is entered.  This change
     preserves compatibility with the Summer '87 INPUT command.  In
     Clipper 5.0 the variable is set to NIL if no new value is entered.
