 47  CHANGED: RIGHT()
--------------------------------------------------------------------------------

     In version 5.0, the standard function RIGHT() was defined in STD.CH
     as an EXTEND.LIB replacement.  This made it unavailable for use
     with the macro (&) operator as well as within index key
     expressions.  In version 5.01 RIGHT() is now included in
     EXTEND.LIB.
