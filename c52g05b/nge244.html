 152 NEW: _storclen()
--------------------------------------------------------------------------------

     _storclen() is a new function in the Extend System that allows you
     to store a character value to a Clipper variable passed by
     reference from Clipper.  With this function, you specify the length
     of the string so that it does not have to be null-terminated.
