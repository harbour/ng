 91  NEW: INIT.CLD
--------------------------------------------------------------------------------

     In addition to any script file called for on the CLD command-line,
     the Debugger automatically searches for a script file with the name
     INIT.CLD.  If a file by this name is located in the current
     directory or anywhere in the DOS PATH, the Debugger executes it as
     a script file.  If both INIT.CLD and a command-line script file are
     present, INIT.CLD is executed first followed by the command-line
     script file.
