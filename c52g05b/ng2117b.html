 142 NEW: Monitor Window
--------------------------------------------------------------------------------

     The Monitor Window is new.  It is used to display monitored
     variables which used to be displayed in the Watch Window.  The
     Monitor remains open on the main Debugger screen as long as
     variables are being monitored.  When the Monitor Window is active,
     variables within the window can be inspected.
