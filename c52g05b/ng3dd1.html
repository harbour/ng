 27  CHANGED: EMPTY()
--------------------------------------------------------------------------------

     EMPTY() has been updated to support array and NIL data types.  For
     example:

     Table: EMPTY() Return Values
     -----------------------------------------------------------------
     Expression      Returns
     -----------------------------------------------------------------
     EMPTY( {} )     .T.
     EMPTY( NIL )    .T.
     -----------------------------------------------------------------
