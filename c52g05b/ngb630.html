 125 NEW: USE
--------------------------------------------------------------------------------

     USE has been enhanced with several new keywords:

     1. NEW opens files in the next available work area.

     2. READONLY opens files with a read-only attribute.

     3. SHARED opens files with shared access regardless of the
        current EXCLUSIVE SETting.
