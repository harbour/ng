 48  CHANGED: SCROLL()
--------------------------------------------------------------------------------

     All parameters to the SCROLL() function are now optional.  If
     parameters are omitted, the dimensions of the visible display are
     used.  If the last parameter (number of lines to scroll) is
     omitted, zero (clear area) is assumed.
