 115 CHANGED: SET EXCLUSIVE*
--------------------------------------------------------------------------------

     SET EXCLUSIVE* is now classified as a compatibility command.  It is
     recommended that you specify the open mode as either SHARED or
     EXCLUSIVE as part of the USE command.
