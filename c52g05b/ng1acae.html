 65  NEW: Get:subscript
--------------------------------------------------------------------------------

     Get:subscript is a new exported instance variable that returns
     information about Gets on array elements.  For a Get object which
     operates on an array element, Get:subscript returns an array of
     numeric values representing the subscripts used in the @...GET
     statement.  If the Get involves a regular variable, Get:subscript
     returns NIL.
