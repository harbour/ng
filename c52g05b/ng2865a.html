 NEW:      READUPDATED()
------------------------------------------------------------------------------

     READUPDATED() is intended primarily for creating new READ Layers.  It is
     identical in functionality to UPDATED() (see the UPDATED() entry in the
     Reference guide), except that it allows the UPDATED() flag to be set.
