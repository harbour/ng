 148 CHANGED: View Workareas Window
--------------------------------------------------------------------------------

     The View Workareas Window replaces the old Status Window.  This
     window is opened using View:Workareas and closed by pressing Esc.
     The View Workareas window displays extensive information regarding
     any database file currently in use.  You select the file you are
     interested in from the left window pane and Tab to the other panes
     to get a closer look.
