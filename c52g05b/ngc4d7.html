 134 NEW: MEMVAR
--------------------------------------------------------------------------------

     MEMVAR is a new statement that declares memory variable names to
     the compiler.  When unaliased references are made to variable names
     declared with MEMVAR, the memory variable alias (MEMVAR->) is
     assumed.
