 90  NEW: Error Messages
--------------------------------------------------------------------------------

     The following new compiler messages have been added in Clipper
     5.01:

     1.  Warning C1007 occurs when a function ends with a statement
         other than RETURN.

     2.  Error C2074 occurs when a #error directive is encountered
         during compilation.

     3.  Error C2086 indicates that a RETURN statement was
         encountered between a BEGIN SEQUENCE statement and the
         corresponding RECOVER statement (or END statement, if no
         RECOVER statement is used).
