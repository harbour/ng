 59  NEW: USE VIA option
--------------------------------------------------------------------------------

     The USE command now supports the VIA <cDriver> clause.  VIA allows
     a replaceable database driver to be specified when a work area is
     used.  If no VIA clause is specified, the default DBFNTX driver is
     used.  This driver implements the standard Clipper database
     behaviors.
