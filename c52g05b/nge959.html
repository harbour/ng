 158 NEW: _xalloc()
--------------------------------------------------------------------------------

     _xalloc() is a new function in the Extend System that allows a C or
     Assembly language function to allocate memory from Clipper's fixed
     heap.  A NULL pointer is returned if the allocation is
     unsuccessful.
