 58  NEW: MEMVARBLOCK()
--------------------------------------------------------------------------------

     MEMVARBLOCK() is a new function that builds a code block designed
     to assign or retrieve a designated memory variable.  When executed
     with an argument, the code block assigns the value of the argument
     to the variable.  When executed without an argument, the code block
     retrieves the current variable value.
