 37  NEW: FIELDWBLOCK()
--------------------------------------------------------------------------------

     FIELDWBLOCK() is a new function that builds a code block designed
     to assign or retrieve a database field in a designated work area.
     The function is identical to FIELDBLOCK() except that it takes a
     work area number in addition to a field name argument.
