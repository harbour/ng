 175 NEW: Environment Variables
--------------------------------------------------------------------------------

     The following new environment variables are supported by various
     components of the Clipper 5.0 system:

   Table: Clipper 5.0 Environment Variables
   -------------------------------------------------------------------------
   Variable    Component            Purpose
   -------------------------------------------------------------------------
   CLIPPERCMD  (compiler)           Set default compiler options
   INCLUDE     (compiler)           Paths to search for header files
   LIB         (linker, runtime)    Paths to search for (.LIB, .PLT, .PLL)
   OBJ         (linker)             Paths to search for object (.OBJ) files
   PLL         (linker, runtime)    Paths to search for .PLT and .PLL files
   RMAKE       (make)               Set default RMAKE command-line options
   TMP         (compiler, linker)   Location for creating temporary files
   RTLINKCMD   (linker)             Set default .RTLink command-line options
   TMP         (compiler, linker)   Location for creating temporary files
   -------------------------------------------------------------------------
