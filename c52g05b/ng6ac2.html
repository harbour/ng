 65  NEW: PAD()
--------------------------------------------------------------------------------

     PADC(), PADL(), and PADR() are new functions that pad a character
     string with a specified fill character.  PADL() pads the original
     string on the left, PADR() pads it on the right, and PADC() pads it
     on both sides so that the original string is centered between the
     fill characters.
