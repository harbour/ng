 CHANGED:  SET ORDER command
------------------------------------------------------------------------------

     The SET ORDER command has been changed to also work with data formats
     other than the traditional Xbase().  The new syntax is:

     SET ORDER TO [<nOrder> | [TAG <cOrderName>]
        [IN <xcOrderBagName>]]
