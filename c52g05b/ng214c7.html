 144 CHANGED: Callstack Window
--------------------------------------------------------------------------------

     To inspect the Callstack Window, simply select the window using the
     Tab or Shift-Tab keys and highlight the item you want to view.  It
     is no longer necessary to press Return to inspect the highlighted
     item.
