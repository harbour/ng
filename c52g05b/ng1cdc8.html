 88  CHANGED: Parameters to Get:postBlock
--------------------------------------------------------------------------------

     In previous versions of Getsys.prg, two parameters were passed to
     Get:postBlock during post-validation: the updated value of the get
     variable, and a logical value representing whether the Get had been
     edited.  In this version, only a single parameter is passed: the
     Get object itself.  This allows more control in the post-validation
     code.
