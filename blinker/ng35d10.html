 Swap / run package functions
------------------------------------------------------------------------------

 The swap / run package functions use the C calling conventions as described
 above in the Blinker Programming Functions section.

 The swap / run functions are provided with CA-Clipper, BASIC and C
 interfaces which are defined in the appropriate language include files
 BLINKER.BI, BLINKER.H, BLINKER.CH, BLINKER.FI, BLINKER.HDR, BLINKER.DEF and
 BLINKER.INC, which can be found in the BAS, C, CLP, FOR, FRC, MOD and PAS
 subdirectories respectively of the Blinker installation directory:
