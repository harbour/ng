 REALMODE                      Specify real mode Windows-based program
------------------------------------------------------------------------------

 Purpose  : Specify a real mode Windows-based program

 Syntax   : REALMODE

 This command specifies that the program can be run under Windows in real or
 protected mode. PROTMODE is assumed by default for EXETYPE WINDOWS.
