 1139: unbalanced DEFBEGINs and DEFENDs
------------------------------------------------------------------------------

 This message indicates that a different number of DEFBEGIN and DEFEND
 commands was found in the module definition file(s), which implies an
 invalid link script file. Please correct and retry.
