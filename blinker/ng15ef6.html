 EXTDICTIONARY                 Enable use of library extended dictionaries if they exist
------------------------------------------------------------------------------

 Purpose  : Enable use of library extended dictionaries if they exist

 Syntax   : EXTDICTIONARY

 Default  : Enabled

 This command specifies that Blinker should use the extended dictionary that
 is appended to library files by some librarian utility programs. The
 extended dictionary is a cross reference of the library contents, and is
 used by Blinker to speed up the processing of libraries. Not all librarians
 provide an extended dictionary - in this case Blinker will process the
 library slightly slower than if it were present.
See Also: NOEXTDICTIONARY
