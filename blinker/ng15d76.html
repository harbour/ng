 ENDAREA                       Specify end of overlay area
------------------------------------------------------------------------------

 Purpose  : Specify the end of an overlay area

 Syntax   : ENDAREA

 The ENDAREA command is used to specify to the linker the end of an overlay
 area, as described in the BEGINAREA section.
