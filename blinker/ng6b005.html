 INCLUDE                       Insert another file into .DEF file
------------------------------------------------------------------------------

 Purpose  : Insert another file into the .DEF file

 Syntax   : INCLUDE filename

 Example  : INCLUDE OTHERFILE

 This command inserts the contents of a specified file into the .DEF file at
 the location of the INCLUDE command. The file to be inserted must contain
 valid module definition file statements. Included files may be nested up to
 4 levels.
