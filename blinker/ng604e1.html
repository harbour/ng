 4.1 New commands
------------------------------------------------------------------------------

 BLINKER HOST PASSUP [ON | OFF]

 This command enables or disables the Blinker extender's automatic pass up of
 IRQ interrupts from real mode to protected mode. This command should only be
 necessary for debugging purposes when testing interrupt handlers, and is
 enabled by default. See also the new /HP+ environment setting.
