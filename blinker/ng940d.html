 BLINKER EXECUTABLE NODELETE   Create .EXE regardless of unresolved externals
------------------------------------------------------------------------------

 Purpose  : Create .EXE regardless of unresolved externals

 Syntax   : BLINKER EXECUTABLE NODELETE

 Default  : Disabled

 This command is used to prevent Blinker from deleting the .EXE file even
 when there are unresolved externals. This can be useful during the
 development cycle, when not all routines have been defined. Note that at
 runtime the program will fail if an unresolved external is called.
