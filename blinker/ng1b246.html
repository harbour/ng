 READONLY                      Specify executable file will be set to read only status.
------------------------------------------------------------------------------

 Purpose  : Specify executable file will be set to read only status.

 Syntax   : READONLY

 Default  : Disabled

 The READONLY command causes Blinker to create the .EXE file with a read only
 file attribute. This command will be of primary interest to developers
 linking and testing a program over a network, as it prevents sharing
 violations when executing the program from several workstations.

 Note that the read only attribute is lost if the file is copied.
