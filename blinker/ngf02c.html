 BLINKER MESSAGE NOBLINK       Disable display of blinking eyes
------------------------------------------------------------------------------

 Purpose  : Disable display of blinking eyes

 Syntax   : BLINKER MESSAGE NOBLINK

 Default  : Disabled

 This command is used to switch off the blinking eyes which are displayed
 while Blinker is running.
