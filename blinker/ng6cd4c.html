 STACKSIZE                     Set stack size in bytes
------------------------------------------------------------------------------

 Purpose  : Set stack size in bytes

 Syntax   : STACKSIZE n

 Example  : STACKSIZE 4096

 The number specified for the stack size must be a positive decimal integer.
