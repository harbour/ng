 DOSSEG                        Use Microsoft DOS segment ordering
------------------------------------------------------------------------------

 Purpose  : Use Microsoft DOS segment ordering

 Syntax   : DOSSEG

 Default  : Disabled

 The DOSSEG command is used to specify that Blinker should arrange the
 segments making up the program in conformance with the segment ordering
 scheme used by the Microsoft high level language compilers.

 Most compilers which require this segment ordering include a DOSSEG request
 in each .OBJ created or in the language library, so this command is not
 needed under normal circumstances.

 Watcom C/C++ requires an explicit DOSSEG command.
