 CODE                          Set default for CODE segment attributes
------------------------------------------------------------------------------

 Purpose  : Set default for CODE segment attributes

 Syntax   : CODE [attribute [, attribute ...] ]

 Example  : CODE PRELOAD EXECUTEREAD

 The default attributes set with the CODE command can be overridden with the
 SEGMENTS command. Each attribute can be specified once in any order. This
 command should be listed before any instances of the SEGMENTS command in the
 definition file that refer to code segments.

 For a complete list of applicable attributes please refer to the SEGMENTS
 command later in this chapter.
