 2.2. CA-Clipper 5.x
------------------------------------------------------------------------------

 The ONLY release versions of CA-Clipper 5.x supported by Blinker 4.0 are
 5.01a, 5.2d/e and 5.3. 5.2c is no longer supported.

 CA-Clipper 5.3 Workbench link templates for real mode, dual mode and
 extended mode programs are provided in the CLP subdirectory in files
 BLIRMD.TPL, BLIDUA.TPL and BLIEXT.TPL respectively. Note that incremental
 linking will default to ON when using these templates.
