 2.1. Borland C++ 4.x
------------------------------------------------------------------------------

 SEARCHing of the Borland C++ libraries or other libraries created with TLIB
 4.x is no longer necessary, nor is the link command NOTABLEOFCONTENTS.
 Blinker 4.0 now fully supports these types of libraries.

 The OBJ subdirectory contains the .OBJ files FP7BCPxx.OBJ which must be
 linked to any protected mode programs which use LIB FP87 instead of LIB EMU
 to force use of the maths coprocessor for floating point operations. These
 version specific files replace the generic BLXBCP87.OBJ from previous
 versions of Blinker.
