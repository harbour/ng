 1124: <file(module)> : first .OBJ not compiled with CA-Clipper
------------------------------------------------------------------------------

 Within an application that contained a CA-Clipper module, the first module
 in a FILE command was not compiled with CA-Clipper. In a CA-Clipper
 application, the first FILE command must refer to the CA-Clipper starting
 program.

 If you receive this message, reorganize your link script so that the first
 FILE statement refers to a CA-Clipper compiled module.
