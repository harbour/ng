 1144: the size of DGROUP plus the heap exceeds 64 kb 
------------------------------------------------------------------------------

 This message indicates that the size of the heap, when added to the size of
 DGROUP including the stack, exceeds 64 kb.

 The heap size may be set or reduced using the module definition file command
 HEAPSIZE nnnn, or may be set to the maximum using HEAPSIZE MAXVAL.
