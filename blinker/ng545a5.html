 1219: BLXCLPnn.OBJ must be linked to the main .EXE
------------------------------------------------------------------------------

 This message indicates that the program attempted to use a CA-Clipper .DLL
 but the appropriate BLXCLPnn.OBJ was not linked to the main .EXE file. This
 message will only occur with CA-Clipper programs.

 Add the command FILE BLXCLPnn, where nn represents the version of CA-Clipper
 being used, at the end of the FILE commands in the link script file for the
 main .EXE.
