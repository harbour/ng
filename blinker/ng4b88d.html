 1159: <filename> is a 32 bit resource file
------------------------------------------------------------------------------

 This message indicates that the specified file name is a 32 bit resource
 file, which cannot be linked to a 16 bit Windows program.

 Use a 16 bit resource compiler to create a 16 bit resource file from the .RC
 file.
