 NODEFLIB                      Ignore default library search records
------------------------------------------------------------------------------

 Purpose  : Ignore default library search records

 Syntax   : NODEFLIB

 Default  : Disabled

 The NODEFLIB command is used to specify that Blinker should ignore default
 library search records which are encountered within object modules. Some
 compilers insert information into each generated object module, specifying
 the names of libraries which the linker may require. Blinker defaults to
 using these records unless NODEFLIB is specified.
