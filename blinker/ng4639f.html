 1114: DEFINE command : '<symbol>' duplicated in <file(module)>
------------------------------------------------------------------------------

 The same information applies as for warning error message number 1004, but
 because of the nature of the duplication, linking cannot continue. To
 resolve this error, either remove the DEFINE command or give the symbol a
 different name and recompile the appropriate .OBJ file.
