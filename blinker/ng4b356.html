 1156: <filename>(<modulename>) is not compatible with 32 bit .EXEs
------------------------------------------------------------------------------

 This message indicates that the specified module in the specified file
 contains code or data which cannot be used in a 32 bit executable. Code must
 be compiled to use 32 bit addresses before it can be used in a 32 bit
 executable.

 If you have the source of the module then recompile it with the correct 32
 bit options for your compiler. If not, contact the supplier of the module
 for a 32 bit version.
