 1208: demonstration date limit exceeded
------------------------------------------------------------------------------

 This message indicates that the program was linked with the BLINKER
 DEMONSTRATION DATE <limit> command to create a demonstration program with an
 expiration date, and that date has now been exceeded. For more details on
 demonstration versions see the related section in Chapter 7.
