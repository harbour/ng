 BLIOVLRES()                   Resume overlay manager operation
------------------------------------------------------------------------------

 Function : Resume overlay manager operation

 Syntax   : BLIOVLRES()

 Return   : Nothing

 This function resumes overlay manager operations following an overlay
 manager suspend call. The overlay manager will reinitialise itself, reopen
 the overlay file(s), and resume processing overlay calls.
See Also: BLIOVLSUS()
