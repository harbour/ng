 4.2 New environment settings
------------------------------------------------------------------------------

 /DI3 (DEBUG INT 3)

 This setting enables the creation of a debug interrupt 3 instruction just
 before the Blinker kernel jumps to the first line of the main program. This
 setting should only be used while debugging, and for more details see the
 file DEBUG.DOC

 /HP+ and /HP- (HOST PASSUP)

 This setting enables or disables the Blinker extender's automatic pass up of
 IRQ interrupts from real mode to protected mode. This setting should only be
 necessary for debugging purposes when testing interrupt handlers, and is
 enabled by default. See also the new BLINKER HOST PASSUP command.

 /WS+ and /WS- (WINDOWS SOFTICE)

 This setting enables or disables checking for the presence of the Soft-Ice/W
 debugger in protected mode running in a Windows enhanced mode DOS box. This
 setting is enabled by default.
