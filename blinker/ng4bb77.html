 1161: more than one NAME or LIBRARY module definition command
------------------------------------------------------------------------------

 This message indicates that multiple NAME or LIBRARY module definition
 commands are specified in the link script or module definition files. Only
 one of a NAME or a LIBRARY module definition command is allowed for each
 module.

 Remove the duplicate or extra module definition commands.
