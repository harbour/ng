 DEFEND                        Specify end of module definition commands
------------------------------------------------------------------------------

 Purpose  : Specify the end of module definition commands

 Default  : None

 Syntax   : DEFEND

 Example  : # Specify program name without a .DEF file
            DEFBEGIN
            NAME TEST
            DEFEND

 This command is used to specify the end of module definition commands
 included in a standard freeformat link script file. This saves maintaining
 two separate link files for DOS extended or Windows programs.

 For more details of the valid module definition commands please refer to
 Chapter 6, Windows Programs.
