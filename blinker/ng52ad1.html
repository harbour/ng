 1209: demonstration time limit exceeded
------------------------------------------------------------------------------

 This message indicates that the program was linked with the BLINKER
 DEMONSTRATION MINUTES <limit> command to create a demonstration program with
 a limited execution time, and that limit has now been reached. For more
 details on demonstration versions see the related section in Chapter 7.
