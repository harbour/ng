 DEFLIB                        Use default library search records
------------------------------------------------------------------------------

 Purpose  : Use default library search records

 Syntax   : DEFLIB

 Default  : Enabled

 The DEFLIB command is used to specify that Blinker should locate and link
 libraries which are specified within an object module. Some compilers insert
 information into each generated object module, specifying the names of
 libraries which the linker may need at link time. This command is enabled by
 default.
See Also: NODEFLIB
