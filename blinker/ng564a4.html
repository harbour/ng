 1302: DPMI function call nnnnh failed with error code nnh
------------------------------------------------------------------------------

 This message indicates that the specified error code was returned from the
 specified DPMI function call.

 Please consult the DPMI documentation for further information on the meaning
 of the function call and the error code.
