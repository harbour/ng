 SWPEMS320()                   Limit swapper to using EMS 3.2 calls
------------------------------------------------------------------------------

 Function : Limit the swapper to using EMS 3.2 calls

 Syntax   : lValue = SWPEMS320(lvalue)

 Return   : The previous setting

 Some offbeat/OEM EMS emulators can sometimes give problems with the
 swapper's use of EMS 4.0 level functions that appear to be improperly
 implemented in the affected drivers. This function can be used to force the
 swapper to only use EMS 3.2 functions.

 Example:

    SWPEMS320(.T.)    // use only EMS 3.2 calls.

 This function will not be needed in most cases.
