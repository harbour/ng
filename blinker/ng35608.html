 Swap / run package functions

 SWPADDENV()                   Increase size of child environment block.
 SWPADDSTR() / SWPADDSTRBAS()  Return string to previous parent program
 SWPCURDIR()                   Enable or disable saving of current directory
 SWPDISMSG()                   Enable or disable displaying of message after swap
 SWPEMS320()                   Limit swapper to using EMS 3.2 calls
 SWPERRLEV()                   Get child process return code
 SWPERRMAJ()                   Get swap system major error code
 SWPERRMIN()                   Get swap system minor error code
 SWPFREEMS()                   Limit use of EMS memory for swapping
 SWPFREXMS()                   Limit use of XMS memory for swapping
 SWPGETKEY()                   Enable / Disable waiting for keypress before restoring parent image.
 SWPGETPID() / SWPGETPIDBAS()  Determine if given program is parent to current program.
 SWPGETSTR() / SWPGETSTRBAS()  Retrieve string sent from child process via SWPSETSTR()
 SWPKEYBRD() / SWPKEYBRDBAS()  Stuff keystrokes into child process
 SWPKEYCLR()                   Alter SWPKEYBRD() behaviour
 SWPNOBOOT()                   Prevent user from rebooting computer
 SWPRUNCMD() / SWPRUNCMDBAS()  Execute child process.
 SWPSETENV()                   Set environment variables for child process
 SWPSETPID() / SWPSETPIDBAS()  Set program ID string for current program.
 SWPSETSTR() / SWPSETSTRBAS()  Set string to return to parent process.
 SWPUSEEMS()                   Enable or disable use of EMS by swap system.
 SWPUSEUMB()                   Enable or disable use of UMBs by swap system.
 SWPUSEXMS()                   Enable or disable use of XMS memory by swap system.
 SWPVIDMDE()                   Enable or disable restoring of video page and mode after swap
