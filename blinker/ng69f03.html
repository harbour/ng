 DESCRIPTION                   Insert descriptive text into program
------------------------------------------------------------------------------

 Purpose  : Insert descriptive text into a program

 Syntax   : DESCRIPTION 'text'

 Example  : DESCRIPTION 'Professional Version 1.00'

 This command is used to embed a string of text up to 255 characters in
 length into a program .EXE or .DLL file. Note that by default the
 information is not used in any way by the created program.

 This command is useful for inserting copyright or version information into a
 program.
