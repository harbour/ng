 1108: stack overflow during linking
------------------------------------------------------------------------------

 This message indicates that the runtime stack for BLINKER.EXE has overflowed
 and should not be displayed under normal circumstances. If this message is
 displayed, please reinstall the Blinker software, and if the error persists
 please contact Blink, Inc. technical support with details of your Blinker
 serial number and system configuration.
