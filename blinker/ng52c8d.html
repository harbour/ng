 1210: overlay has been prematurely freed
------------------------------------------------------------------------------

 This message will only occur when using CA-Clipper Summer '87 and the
 dynamic allocation method for overlays. This message indicates that the
 memory assigned to an overlay has been prematurely freed by the CA-Clipper
 memory management routines, which can happen under very tight memory
 conditions.

 This is a CA-Clipper Summer '87 error which the Blinker overlay manager
 checks for whenever it manipulates dynamic memory.

 Do not confuse this message with the CA-Clipper 5.x internal error message
 1210.
