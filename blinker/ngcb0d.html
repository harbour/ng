 BLINKER LINK VERSION          Specify minimum Blinker version required
------------------------------------------------------------------------------

 Purpose  : Specify minimum Blinker version required

 Syntax   : BLINKER LINK VERSION <n.n>

 Default  : 0.0

 Example  : # Blinker 2.0 required to process this script
            BLINKER LINK VERSION 2.0

 This command is used to ensure a minimum version of Blinker to process the
 link script. If the version of Blinker is less than the specified minimum
 version number, Blinker will terminate with an error. Versions of Blinker
 prior to Blinker 2.0 do not recognize this command!
