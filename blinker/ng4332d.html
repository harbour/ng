 1012: segments in GROUP <groupname> have conflicting attributes
------------------------------------------------------------------------------

 This message indicates that segments in the named group have different
 attribute settings in SEGMENTS directives in the module definition file.
 Segments in the same group are stored as one segment so must all have the
 same attributes.

 The last setting found in the definition file will be the one used.
