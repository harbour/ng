 1173: an undetermined EMS error has occurred
------------------------------------------------------------------------------

 The 1173 error can be avoided temporarily by turning off Blinker's link time
 use of EMS with the BLINKER LINK EMS OFF command until the source of the EMS
 problem can be resolved. This assists in identifying the problem but is not
 recommended as a resolution.
