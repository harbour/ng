 # (comment)                   A script file comment
------------------------------------------------------------------------------

 Purpose  : A script file comment

 Syntax   : # ...

 Example  : # To link or not to link, that is the question.

 The remainder of all lines in the script file containing the character # are
 treated as comments and are ignored by Blinker.
