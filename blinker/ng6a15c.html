 EXETYPE                       Identify target operating system 
------------------------------------------------------------------------------

 Purpose  : Identify the target operating system

 Syntax   : EXETYPE [ WINDOWS [n.nn] | OS2 | DOS ]

 Example  : EXETYPE WINDOWS 3.1

 This command specifies the operating system under which the program is to be
 run.

 When creating Windows programs, the version of Windows is important to
 ensure that the program will be able to run.

 The default EXETYPE is WINDOWS, and the default Windows version number is
 3.00.
