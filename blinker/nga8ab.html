 BLINKER HOST MESSAGE          Enable / disable extender's host display.
------------------------------------------------------------------------------

 Purpose  : Enable / disable the extender's host display.

 Syntax   : BLINKER HOST MESSAGE ON | OFF

 Default  : OFF (disable host messages)

 This command can be used to enable the display of various informational
 messages concerning the available hosts at program startup.

 The command provides a convenient method of checking the current host,
 Blinker version number, processor type and amount of memory available when
 an extended or dual mode program starts execution:

 For example:

    BLX286 : EMS/XMS sharing present
    BLX286 : DPMI host not detected
    BLX286 : XMS host detected
    BLX286 : VCPI host detected
    BLX286 : DPMI host emulator initialised
    BLX286 : Active host is VCPI (v3.20 i486 5668 kb)

 Note also that the BLINKER environment variable setting /HMc will override
 this setting.
