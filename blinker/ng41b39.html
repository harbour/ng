 1001: ignoring invalid link script file command parameter in '<command>'
------------------------------------------------------------------------------

 This message indicates that the link script file command specified has been
 ignored as it contains an invalid parameter value.

 See Chapter 9, Blinker Commands for details on the parameters required by
 the command displayed.
