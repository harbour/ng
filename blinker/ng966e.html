 BLINKER EXECUTABLE REAL       Specify linking of real mode executable
------------------------------------------------------------------------------

 Purpose  : Specify the linking of a real mode executable

 Syntax   : BLINKER EXECUTABLE REAL

 Example  : # Create a real mode only program
            BLINKER EXECUTABLE REAL

 This command specifies that Blinker should produce a real mode executable,
 and is the default if no other executable type is specified.

 Real mode executables may optionally contain dynamic overlays, but make no
 use of protected mode, so will run on all 80x86 processors.
