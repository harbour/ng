 1172: an undetermined XMS error has occurred
------------------------------------------------------------------------------

 The 1172 error can be avoided temporarily by turning off Blinker's link time
 use of XMS with the BLINKER LINK XMS OFF command until the source of the XMS
 problem can be resolved. This assists in identifying the problem but is not
 recommended as a resolution.
