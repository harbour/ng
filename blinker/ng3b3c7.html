 SWPNOBOOT()                   Prevent user from rebooting computer
------------------------------------------------------------------------------

 Function : Prevent the user from rebooting the computer

 Syntax   : lValue = SWPNOBOOT(lFlag)

 Return   : The previous setting

 Default  : Disabled

 This function is used to enable or disable trapping of the <CTRL><ALT><DEL>
 keystroke sequence while the kernel is resident during the swap, preventing
 the user from soft rebooting while a program is swapped out. Please note
 that this mechanism does not function under DesqView or similar environments
 which hook the keyboard vector and look for this key sequence, since such
 software intercepts the keystroke before it gets to the swap kernel.
