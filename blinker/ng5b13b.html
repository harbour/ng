 Swap / Run package settings
------------------------------------------------------------------------------

 /SD<drive:\path> (swap path)

 This setting specifies the path where the temporary file created by the
 swapper will be created. See the function SWPRUNCMD() for more details.

 /SE+ and /SE- (use of EMS memory)

 This setting enables and disables the use of EMS memory by the swap package.
 See the function SWPUSEEMS() for more details.

 /SU+ and /SU- (use of upper memory blocks)

 This setting enables and disables the use of upper memory blocks by the swap
 package. See the function SWPUSEUMB() for more details.

 /SX+ and /SX- (use of XMS memory)

 This setting enables and disables the use of XMS memory by the swap package.
 See the function SWPUSEXMS() for more details.
