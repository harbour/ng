 NOPACKDATA                    Disable packing of NEW EXE data segments
------------------------------------------------------------------------------

 Purpose  : Disable packing of NEW EXE data segments

 Syntax   : NOPACKDATA

 Default  : Enabled

 This command specifies that Blinker should not pack data segments when
 creating NEW exe format executables (Windows and DOS extended programs.)

 Blinker defaults to not packing data segments when creating NEW EXE format
 programs.
