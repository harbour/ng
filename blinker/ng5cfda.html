 Periscope and Periscope/32
------------------------------------------------------------------------------

 Overview

 Blinker 3.2 includes full support for the Periscope debugger for real mode
 programs and the Periscope/32 debugger for real, protected, dual and Windows
 programs using two machines. Periscope/32 requires one machine to run the
 program being debugged and a second machine, on which the source code is
 stored, to run the debugger itself.

 To use Soft-Ice/W with DOS real mode or protected mode programs simply link
 for CODEVIEW with CVV 2 or 5, run TS /e xxxx and then run the program
 xxxx.EXE directly or RUN xxxx.EXE.
