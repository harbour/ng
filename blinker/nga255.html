 BLINKER EXTMEM LIMIT          Specify total extended memory to be allocated by the
------------------------------------------------------------------------------

 Purpose  : Specify the total extended memory to be allocated by the
            extender

 Syntax   : BLINKER EXTMEM LIMIT <nuSize>

 Default  : All available extended memory

 This command specifies the total amount of physical memory the extender will
 allocate before the extender's VM system begins to swap to disk in order to
 satisfy new allocation requests. This command can be used to ensure that a
 DOS extended program does not use up all the available memory unnecessarily,
 or to simulate running in lower memory conditions.

 Note also that the BLINKER environment variable setting /ELnnnn will
 override this function.
