 APPLOADER                     Replace default Windows loader with custom loader
------------------------------------------------------------------------------

 Purpose  : Replace the default Windows loader with a custom loader

 Syntax   : APPLOADER 'loadername'

 Example  : APPLOADER 'MYLOADER'

 This optional command tells Blinker to use an externally defined loader as
 the identifying portion of the loader code segment name by prefixing it with
 LOADER_.

 The command is intended to support self-loading FORTRAN programs, and must
 appear before any SEGMENTS commands.
