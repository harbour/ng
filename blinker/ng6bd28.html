 PROTMODE                      Specify protected mode Windows-based program
------------------------------------------------------------------------------

 Purpose  : Specify a protected mode Windows-based program

 Syntax   : PROTMODE

 This command specifies that the program should only run under Windows in
 protected mode. This command is assumed if EXETYPE has been set to WINDOWS.
