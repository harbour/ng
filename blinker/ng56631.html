 1303: DOS function call nnh is not supported
------------------------------------------------------------------------------

 This message indicates that the program attempted to use the specified DOS
 function call, and that call is not supported by the DOS extender. If this
 is not the expected behaviour please contact Blink, Inc technical support
 with your Blinker serial number and details of the problem.
