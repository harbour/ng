 NOTABLEOFCONTENTS             Disable processing of library table of contents
------------------------------------------------------------------------------

 Purpose  : Disable processing of library table of contents

 Syntax   : NOTABLEOFCONTENTS

 Default  : Disabled

 This command forces Blinker to SEARCH all libraries, i.e. ignore all the
 library table of contents. In certain unusual circumstances this command can
 be used to emulate the order in which Borland's TLINK linker processes
 modules within a library.

 This command applies globally to all libraries processed during the link.
