 1314: unable to create file <filename>
------------------------------------------------------------------------------

 This message indicates that the DOS extender virtual memory system was
 unable to create the specified temporary swap file.

 The most likely causes of this error message are that there is already an
 existing file of the same name which has been assigned read only status, or
 no create access rights have been granted to the destination drive or
 directory, or that the destination disk is full.
