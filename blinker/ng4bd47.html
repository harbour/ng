 1162: <filename> should not be linked to a Windows program
------------------------------------------------------------------------------

 This message indicates that the specified file is a Blinker file
 specifically for use with the DOS extender, so should not be linked to a
 Windows program.

 Remove the FILE or LIBRARY command referring to the specified file.
