 1171: .EXE file size limit exceeded
------------------------------------------------------------------------------

 This message indicates that the .EXE file exceeded Blinker's internal size
 limit. Please contact Blink, Inc technical support with your Blinker serial
 number and exact details of the circumstances causing the error.
