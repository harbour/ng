 BATCH                         Specify to linker it is running in batch mode
------------------------------------------------------------------------------

 Purpose  : Specify to the linker it is running in batch mode

 Syntax   : BATCH

 Default  : Enabled

 The BATCH command is used to specify to the linker that it is being run in
 batch mode. If a particular file or library cannot be found in the
 appropriate path then linking terminates with an error, rather than allowing
 an interactive alternative. Blinker ignores the BATCH command as it is the
 default mode of operation for Blinker.
