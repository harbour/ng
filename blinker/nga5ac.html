 BLINKER HOST DPMI             Enable / disable extender's use of DPMI host.
------------------------------------------------------------------------------

 Purpose  : Enable / disable the extender's use of a DPMI host.

 Syntax   : BLINKER HOST DPMI ON | OFF

 Default  : ON (enable use of DPMI)

 This command can be used to disable the extender's use of a DPMI host for
 testing under non-DPMI hosts, or to disable use of a DPMI host if problems
 are suspected.

 Under normal circumstances, the extender checks for hosts in the following
 order:

    DPMI
    VCPI
    XMS

 Note also that the BLINKER environment variable setting /HDc will override
 this setting.
See Also: BLINKER HOST QDPMI BLINKER HOST VCPI BLINKER HOST XMS
