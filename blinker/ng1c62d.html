 UPPERCASE                     Convert all symbols to uppercase
------------------------------------------------------------------------------

 Purpose  : Convert all symbols to uppercase

 Syntax   : UPPERCASE

 Default  : Enabled

 The UPPERCASE command is used to specify to the linker that all symbols
 should be converted to upper case before being added to the symbol table.
 Blinker defaults to uppercase.
See Also: MIXCASE
