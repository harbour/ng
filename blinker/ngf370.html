 BLINKER MESSAGE WINK          Enable single eye winking during linking
------------------------------------------------------------------------------

 Purpose  : Enable single eye winking during linking

 Syntax   : BLINKER MESSAGE WINK [LEFT]

 Default  : Right winking eye

 Example  : # Display a winking right eye
            BLINKER MESSAGE WINK

 This command is used to enable the winking of only one eye while Blinker is
 running. If no parameter is entered, the right eye will wink (from Blinker's
 point of view, i.e. from inside the screen looking out). The parameter LEFT
 causes the left eye to wink.
See Also: BLINKER MESSAGE NOBLINK
