 1004: DEFINE command : '<symbol>' duplicated in <filename(modulename)>
------------------------------------------------------------------------------

 This message indicates that the symbol <symbol> defined as public in a
 DEFINE command is also defined as public in the second module.

 This will occur if a symbol in an .OBJ or .LIB file has the same name as one
 in a DEFINE command, but the symbol in the .OBJ or .LIB file will be
 ignored.
