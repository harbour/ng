 SWPVIDMDE()                   Enable or disable restoring of video page and mode after swap
------------------------------------------------------------------------------

 Function : Enable or disable restoring of the video page and mode after the
            swap

 Syntax   : lValue = SWPVIDMDE(lFlag)

 Return   : The previous setting

 Default  : Disabled

 When enabled, this function causes the swap kernel to save the video mode
 and active page prior to the swap, and restore them following the swap. Note
 that the contents of the video buffer are not saved.
