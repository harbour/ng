 1121: BASIC requires the first .OBJ file to be in the root
------------------------------------------------------------------------------

 Microsoft BASIC requires the main .OBJ file of a program to be in the root,
 i.e. outside any BEGIN/ENDAREAs. All other BASIC .OBJs can be overlaid.
