 BLINKER HOST PASSUP           Enable / disable automatic pass up of IRQ interrupts.
------------------------------------------------------------------------------

 Purpose  : Enable / disable automatic pass up of IRQ interrupts.

 Syntax   : BLINKER HOST PASSUP ON | OFF

 Default  : ON (enable passup of IRQ interrupts)

 This command can be used to enable or disable the extender's automatic pass
 up of IRQ interrupts from real mode to protected mode.

 This command should only be necessary for debugging purposes when testing
 interrupt handlers.

 Note also that the BLINKER environment variable setting /HPc will override
 this setting.
