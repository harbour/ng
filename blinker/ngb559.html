 BLINKER HOST XMS              Enable / disable extender's use of XMS host.
------------------------------------------------------------------------------

 Purpose  : Enable / disable the extender's use of an XMS host.

 Syntax   : BLINKER HOST XMS ON | OFF

 Default  : ON (enable use of XMS)

 This command can be used to disable the extender's use of an XMS host for
 testing under a non XMS host, or to disable use of an XMS host if problems
 are suspected.

 Under normal circumstances, the extender checks for and makes use of hosts
 in the following order:

    DPMI
    VCPI
    XMS

 Note also that the BLINKER environment variable setting /HXc will override
 this setting.
See Also: BLINKER HOST DPMI BLINKER HOST QDPMI BLINKER HOST VCPI
