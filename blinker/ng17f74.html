 MIXCASE                       Specify case sensitivity
------------------------------------------------------------------------------

 Purpose  : Specify case sensitivity

 Syntax   : MIXCASE [EXPORT]

 Default  : Disabled

 The MIXCASE command causes Blinker to treat all publics and externals as
 case sensitive at link time, rather than converting them to UPPERCASE. This
 command is of primary interest to developers using C, who may require case
 sensitivity.

 By default all export definition names in module definition files are
 converted to uppercase when they are processed. When MIXCASE EXPORT is
 specified in the link script file the export names are treated as case
 sensitive.
