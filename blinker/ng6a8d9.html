 FUNCTIONS                     Specify function order and location
------------------------------------------------------------------------------

 Purpose  : Specify function order and location

 Syntax   : FUNCTIONS [:segmentname] functionnames

 Example  : FUNCTIONS :seg1
            func1
            func2
            func3

 This command places functions contained in COMDAT records in a specified
 physical order, and can optionally assign them to a specified segment name.
