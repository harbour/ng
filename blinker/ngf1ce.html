 BLINKER MESSAGE NOWARNING     Disable display of link time warning messages
------------------------------------------------------------------------------

 Purpose  : Disable display of link time warning messages

 Syntax   : BLINKER MESSAGE NOWARNING

 Default  : Disabled

 This command is used to suppress the display of Blinker link time warning
 messages 1001 to 1100.
