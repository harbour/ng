 BLINKER EXECUTABLE NOBLINK    Causes NO Blinker code whatsoever to be linked in
------------------------------------------------------------------------------

 Purpose  : Causes NO Blinker code whatsoever to be linked in

 Syntax   : BLINKER EXECUTABLE NOBLINK

 Default  : Disabled

 This command causes Blinker to ignore all overlays, demo restrictions, the
 CA-Clipper 5.x paging system, and will give unresolved externals for all
 Blinker functions including the swap functions. This is mainly used in order
 to create small C / ASM programs with no overhead.

 DO NOT USE FOR CA-CLIPPER EXCEPT FOR SMALL PROGRAMS WITHOUT OVERLAYS !!
