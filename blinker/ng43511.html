 1013: ignoring invalid module definition file command parameter in '<command>'
------------------------------------------------------------------------------

 This message indicates that the module definition file command specified has
 been ignored as it contains an invalid parameter value.

 See Chapter 8, Windows Programs for details on the parameters required by
 the command displayed.
