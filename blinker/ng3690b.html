 SWPCURDIR()                   Enable or disable saving of current directory
------------------------------------------------------------------------------

 Function : Enable or disable saving of the current directory

 Syntax   : lValue = SWPCURDIR(lFlag)

 Return   : The previous setting

 Default  : Enabled

 Enable or disable the restoring of the current drive and directory following
 the swap. When enabled, the swap kernel will restore the default drive and
 current directory following the swap.
