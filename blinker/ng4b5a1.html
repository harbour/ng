 1157: Importing from DLLs is not supported in dual mode
------------------------------------------------------------------------------

 This message indicates that an attempt was made to import a symbol from a
 DOS extended DLL into a dual mode program, which is not supported.

 Relink the program as an extended mode program in order to use DLLs.
