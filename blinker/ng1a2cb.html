 NWIDTH                        Specify width of symbol names in map file.
------------------------------------------------------------------------------

 Purpose  : Specify the width of symbol names in the map file.

 Syntax   : NWIDTH <nuWidth>

 Default  : 23

 This command specifies the maximum length of a symbol name which will be
 output to the map file. Symbol names which are longer than the NWIDTH value
 will be truncated.
