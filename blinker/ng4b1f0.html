 1155: portable .EXE file needs larger file alignment value
------------------------------------------------------------------------------

 This message indicates that the portable .EXE file is too large for the
 current file alignment size.

 Use the BLINKER EXECUTABLE ALIGNMENT command to increase the file alignment
 size.
