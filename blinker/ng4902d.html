 1132: error deleting file <filename>
------------------------------------------------------------------------------

 This message indicates that either the file specified is marked as read only
 or is still in use, or that network delete access is not currently granted,
 or that the file exists in an alternate directory specified with the DOS
 APPEND command or network search command.
