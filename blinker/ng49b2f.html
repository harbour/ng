 1140: new .EXE contains more than 254 segments
------------------------------------------------------------------------------

 This message indicates that the new .EXE file would contain more than the
 maximum of 254 segments. This should only occur if the BLINKER SEGMENT
 THRESHOLD command is used to reduce the size of combined segments, so the
 THRESHOLD should be increased until the .EXE is created successfully.
