 BLINKER HOST VCPI             Enable / disable extender's use of VCPI host.
------------------------------------------------------------------------------

 Purpose  : Enable / disable the extender's use of a VCPI host.

 Syntax   : BLINKER HOST VCPI ON | OFF

 Default  : ON (enable use of VCPI)

 This command can be used to disable the extender's use of a VCPI host for
 testing under a non VCPI host, or to disable use of a VCPI host if problems
 are suspected.

 Under normal circumstances, the extender checks for hosts in the following
 order:

    DPMI
    VCPI
    XMS

 Note also that the BLINKER environment variable setting /HVc will override
 this setting.
See Also: BLINKER HOST DPMI BLINKER HOST QDPMI BLINKER HOST XMS
