 1129: BLINKER OVERLAY DYNAMIC is only applicable to CA-Clipper Summer '87
------------------------------------------------------------------------------

 Blinker supports the dynamic overlay allocation method (whereby memory for
 overlays is allocated from Clipper's free pool) for CA-Clipper Summer '87
 only, and the application being linked is not a CA-Clipper Summer '87
 application.

 Remove the BLINKER OVERLAY DYNAMIC command from your link script.
