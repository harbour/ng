 BLINKER HOST QDPMI            Enable / disable extender's use of QDPMI host.
------------------------------------------------------------------------------

 Purpose  : Enable / disable the extender's use of a QDPMI host.

 Syntax   : BLINKER HOST QDPMI ON | OFF

 Default  : ON (enable use of QDPMI)

 This command can be used to disable the extender's use of a QDPMI host for
 testing under non-QDPMI hosts, or to disable use of a QDPMI host if problems
 are suspected.

 Under normal circumstances, the extender checks for hosts in the following
 order:

    DPMI
    VCPI
    XMS

 Note also that the BLINKER environment variable setting /HDc will override
 this setting.
See Also: BLINKER HOST DPMI BLINKER HOST VCPI BLINKER HOST XMS
