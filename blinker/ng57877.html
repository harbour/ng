 1312: insufficient extended memory available to run 
------------------------------------------------------------------------------

 This message indicates that there is insufficient extended memory available
 to load and run the program. The amount of memory to run the program will
 typically depend on the size of the program, but a minimum may have been set
 using the BLINKER EXECUTABLE EXTENDED nnnn command at link time.
