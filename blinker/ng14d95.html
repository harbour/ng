 DEFFILE                       Specify module definition command file
------------------------------------------------------------------------------

 Purpose  : Specify module definition command file

 Default  : None

 Syntax   : DEFFILE <filename>

 Example  : # Specify module definition file TEST
            DEFFILE TEST

 This command is used to specify the name of a module definition command file
 in a standard freeformat link script file. A default file extension of .DEF
 is assumed if none is specified.

 Processing of the link script file will be suspended while the module
 definition file specified (and any nested files therein) are processed.
 Processing will then continue with the next line of the link script file.

 For more details of the valid module definition commands please refer to
 Chapter 6, Windows Programs.
