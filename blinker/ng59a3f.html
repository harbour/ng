 Runtime environment settings
------------------------------------------------------------------------------

 Various aspects of Blinker linked programs may be overridden at program
 runtime through the use of BLINKER environment variable (the name of which
 may be changed using the BLINKER ENVIRONMENT NAME command at link time).

 All BLINKER environment settings begin with a '/' character, and should be
 separated by spaces eg:

 SET BLINKER=/HM+ /EL1000
