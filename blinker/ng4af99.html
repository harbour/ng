 1154: portable .EXE file contains more than 128 sections
------------------------------------------------------------------------------

 This message indicates that the portable .EXE or .DLL file contains more
 than the maximum of 128 sections.

 Please consult your compiler documentation in order to reduce the number of
 sections in the .EXE. One way to identify the section names would be to
 continue to comment out or remove .OBJ files from the link script and relink
 until an EXE file can be created, and then examine the MAP file to find the
 section names.
