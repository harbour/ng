 1176: invalid symbol token
------------------------------------------------------------------------------

 Blinker has encountered an invalid CA-Clipper symbol token when compressing
 the symbol table of a CA-Clipper application. The most likely cause of this
 is corruption of one or more object modules or libraries.

 Recompile your program and link with VERBOSE mode enabled in order to
 determine the module causing the error.
