 BLIMEMSIZ()                   Get size of CA-Clipper Summer '87 dynamic memory area
------------------------------------------------------------------------------

 Function : Get the size of the CA-Clipper Summer '87 dynamic memory area

 Syntax   : nlValue = BLIMEMSIZ()

 Return   : Total size of the free pool in bytes

 This function returns the total size in bytes of the dynamic memory area.
 Dynamic memory is the memory used by CA-Clipper to store all temporary
 values of variables and buffers while an application is running.

 Note that the CA-Clipper Summer '87 free pool is allocated from DOS on an
 'as-needed' basis, and unless MEMORY(0), BLIMEMPAK(-1) or BLIMEMBLK() are
 executed (which cause CA-Clipper to allocate all available memory to the
 free pool from DOS), the value returned by this function will reflect only
 the currently allocated free pool.

 Example:

    * Get the total free pool size
    *
    ? "The total size of CA-Clipper's free pool is"
    ?? BLIMEMSIZ(), "bytes"
