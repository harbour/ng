 Overlay Manager Settings
------------------------------------------------------------------------------

 /CEnnn,nn (EMS overlay cache size) 

 This setting controls the size of the EMS overlay cache. See the BLINKER
 CACHE EMS command for more details.

 /CXnnn,nn (XMS overlay cache size)

 This setting controls the size of the XMS overlay cache. See the BLINKER
 CACHE XMS command for more details.

 /DHn (debug handle)

 This setting specifies the file handle to be used as the destination of
 debug output from BLINKER. See the BLINKER DEBUG HANDLE command for more
 details.

 /DI3 (debug interrupt 3)

 This setting (DEBUG INT 3) enables the creation of a debug interrupt 3
 instruction just before the Blinker kernel jumps to the first line of the
 main program. This setting should only be used while debugging, and for more
 details see the file DEBUG.DOC

 /OOnnn (overlay operating size)

 This setting controls the size of the BLINKER overlay pool. See the BLINKER
 OVERLAY OPSIZE command for more details.

 /OP+ and /OP- (overlay pageframe)

 This setting enables and disables the use of the EMS pageframe for the
 overlay pool. See the BLINKER OVERLAY PAGEFRAME command for more details.

 /OU+ and /OU- (overlay UMB)

 This setting enables and disables the use of UMBs for the overlay pool. See
 the BLINKER OVERLAY UMB command for more details.
