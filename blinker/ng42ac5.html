 1008: <filename> : invalid incremental link file
------------------------------------------------------------------------------

 This message indicates that the previous incremental link was unsuccessful
 and left an invalid incremental link file, or that the file specified with
 the BLINKER INCREMENTAL FILE command was not a valid incremental link file
 for the program concerned. A full link will be performed to recreate the
 program .EXE and its associated incremental link file.
