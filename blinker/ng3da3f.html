 SWPUSEXMS()                   Enable or disable use of XMS memory by swap system.
------------------------------------------------------------------------------

 Function : Enable or disable use of XMS memory by the swap system.

 Syntax   : lValue = SWPUSEXMS(lFlag)

 Return   : The previous setting

 Default  : Enabled

 This function enables or disables the use of XMS memory for swap image
 storage. The return value is the previous setting. If XMS use is enabled,
 the parent image can be saved (and restored) much more quickly that if it
 were written to disk.

 Note that when both XMS and EMS memory are available, the swap system will
 use XMS first. Note also that the BLINKER environment variable setting /SXc
 will override this function.
