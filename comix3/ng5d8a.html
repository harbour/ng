cmRestFilter()        Restore a saved filter       
------------------------------------------------------------------------------

Synopsis

    cmRestFilter(filtSave) -> NIL

Arguments

    filtSave is a "saved filter" which has previously been obtained by
    calling cmSaveFilter().

Returns

    NIL.

Description

    Restores the filter state using a saved version (previously set up by
    cmSaveFilter()).

    This allows you to stack filters, or swap filters without having to go
    through the work which was spent in executing the original cmFilter().

    cmRestFilter() will restore saved versions of:

    o   The Filter Condition
    o   The Filter RecordList
    o   The Current Record Number

    Please note that cmRestFilter() "destroys" the saved filter.  You can
    only restore a saved filter once.

Example

    use demo index last, age
    cmFilter("age = 30")                && Restrict to the 30's
    saveFilt := cmSaveFilter()          && Save the filter
    cmFilter("age = 31")                && Set up a new filter
    ? dbFilter()                        && "age = 31"
    list last, age                      && All the 31's in last order
    cmRestFilter(saveFilt)              && Restore the previous filter
    ? dbFilter()                        && "age = 30"
    list last, age                      && All the 30's in last order

Example

    function tempQuery()
    *   Save the current filter
    local saveFilt := cmSaveFilter()
    local newQuery := space(60)

    *   Get the new condition
    clear
    @ 0, 0 say "Condition: " get newQuery
    read

    *   Set up the new condition and let user browse
    cmFilter(newQuery)
    dbedit()

    *   Restore things as they were
    cmRestFilter(saveFilt)
    return
