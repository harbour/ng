cmKeyNo()             Return the logical key number (filtered)
------------------------------------------------------------------------------

Synopsis

    cmKeyNo( [nOrder] | [cTag [, cBag]] ) -> nKeyNo

Arguments

    nOrder, cTag and cBag are optional parameters which specify the index.
    If omitted, the current master index will be used.

Returns

    The logical key number obeying the filter condition (and any index scope
    in effect).

Description

    cmKeyNo() is similar to cmxKeyNo(), but returns the logical key number
    of records which meet the filter condition.  (cmxKeyNo() returns the
    "raw" count regardless of the filter).

    cmKeyNo() will force Linear Optimization to be performed so that the
    return value is accurate.

    cmKeyNo() obeys the current index scope (if any).  cmKeyNo() will return
    1 for the first record which meets the filter condition and falls within
    the current index scope, etc..

Example

    index on age to age
    index on last to last
    use demo index age, last

    go top
    ? "cmxKeyNo(): ", cmxKeyNo()
    ? " cmKeyNo(): ", cmKeyNo()             && Will return same (no filter)

    cmFilter("age >= 25")
    go top
    skip 1
    ? "cmxKeyNo(): ", cmxKeyNo()            && Who knows?
    ? " cmKeyNo(): ", cmKeyNo()             && Will return 2
