INDEX                Create an index
------------------------------------------------------------------------------

    See the Comix Command Reference for complete information on this
    command.

    When using ClipMore, the FOR condition clause will automatically be
    optimized when creating conditional indexes.

    ClipMore will NOT optimize the FOR condition, however, if a WHILE,
    USECURRENT or NOOPTIMIZE clause is used.
