rlSet()               Add a record to (set bit in) a RecordList
------------------------------------------------------------------------------

Synopsis

    rlSet(rl, nRecno) -> NIL

Arguments

    rl is the RecordList to which the record is to be added.

    nRecno is the record number to add to the RecordList.

Returns

    NIL.

Description

    rlSet() allows you to add a record to a RecordList (set a bit in the
    RecordList).

Example

    rl := rlNew(50)
    ? rlCount(rl)               && Will return 0
    rlSet(rl, 30)               && Add record 30
    rlSet(rl, 40)               && Add record 40
    ? rlCount(rl)               && Will now return 2

See Also

    rlClear(), rlTest()
