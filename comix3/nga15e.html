rlLen()               Get the length of a RecordList
------------------------------------------------------------------------------

Synopsis

    rlLen(rl) -> nLength

Arguments

    rl is the RecordList for which the length is to be determined.

Returns

    rlLen() returns the length of the RecordList.

Description

    rlLen() returns the length of the RecordList (i.e., the maximum number
    of records which can be stored in the RecordList).

Example

    rl := rlNew(50)
    ? rlLen(rl)             && Will return 50
    ? rlCount(rl)           && Will return 0 (no recs in RecordList)
    rlDestroy(rl)

Example

    use demo
    rl := rlNew()
    ? rlLen(rl)             && Will return number of recs in database
    ? recCount()            && Same as above
    rlDestroy(rl)
