rlNewQuery()          Create a RecordList of recs meeting the condition
------------------------------------------------------------------------------

Synopsis

    rlNewQuery(cQuery) -> rlResult

Arguments

    cQuery is the condition which records should meet (as a character
    string).

Returns

    rlNewQuery() returns a RecordList of records which meet the condition by
    scanning available records.  This RecordList may contain Maybe's.

Description

    rlNewQuery() analyzes the condition, matches up parts of the condition
    with indexes and determines which records meet the condition by scanning
    indexes.

    This is a very powerful function in that it handles matching up parts of
    the condition with indexes automatically.

    However, unlike other RecordList functions, this function can return a
    RecordList which contains Maybe's (see the description under
    cmDoLinear()).

    You can use rlMaybeEval() to clear off any Maybe's (and make them Yes or
    No).

    rlOptLevel() will return the level of optimization achieved by this
    function (full optimization, partial optimization, or no optimization).

Example

    use demo index age
    rl := rlNewQuery("age >= 25 .and. age <= 30")
    ? "Records meeting condition: "
    curRec := 0
    do while ((curRec := rlNextRecno(curRec)) != 0)
        ? curRec
    enddo
    rlDestroy(rl)

Example

    *   Use demo (only index is age)
    use demo index age

    *   No index on salary, so can't optimize that"
    rl := rlNewQuery("age < 25 .and. salary <= 25000")
    ? "Approximate count: ", rlCount(rl)

    *   Clear off the Maybe's
    rlMaybeEval(rl, {|| age < 25 .and. salary <= 25000})

    *   Now give the exact count
    ? "      Exact count: ", rlCount(rl)
