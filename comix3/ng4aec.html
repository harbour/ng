cmKeyGoto()           Goto the specified logical key number (filtered)
------------------------------------------------------------------------------

Synopsis

    cmKeyGoto(nKeyno) -> NIL

Arguments

    nKeyno is the number of the key to go to.

Returns

    NIL.

Description

    cmKeyGoto() goes to the n'th filtered key.  This is useful when using
    scroll bars, and you want to quickly get to the n'th key when the user
    clicks on a certain position on the scroll bar.

    cmKeyGoto() obeys the current index scope (if any).  For example,
    cmKeyGoto(1) will go to the first record which meets the condition _and_
    falls within the index scope.

Example

    index on age to age
    index on last to last
    use demo index age, last

    cmFilter("last = 'J'")
    cnt := cmFiltCount()

    *   Go to the halfway point in the list
    cmKeyGoto(cnt / 2)
