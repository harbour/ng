cmDestroyFilter()     Destroy a saved filter without restoring it
------------------------------------------------------------------------------

Synopsis

    cmDestroyFilter() -> NIL

Arguments

    None.

Returns

    NIL.

Description

    cmDestroyFilter() allows you to discard a saved filter which has
    previously been constructed by cmSaveFilter().

    All saved filters constructed by cmSaveFilter() must be either restored
    (using cmRestFilter()), or destroyed (using this function).

Example

    use demo index age
    cmFilter("age = 30")                && Restrict to the 30's
    saveFilt := cmSaveFilter()          && Save the filter
    cmFilter("age = 31")                && Set up a new filter
    ? dbFilter()                        && "age = 31"
    list age                            && All the 31's

    *   Ask user if we want to restore the previous fitler
    goBack := .f.
    @ 0, 0 say "Go back? " get goBack
    read

    *   If we want to go back
    if (goBack)
        cmRestFilter(saveFilt)          && Restore the previous filter
        ? dbFilter()                    && "age = 30"
        list age                        && Will now be all the 30's
    else
        cmDestroyFilter(saveFilt)       && Just discard the saved filter
    endif

See Also

    cmRestFilter(), cmSaveFilter()
