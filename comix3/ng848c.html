rlCount()             Count the number of records in a RecordList
------------------------------------------------------------------------------

Synopsis

    rlCount(rl) -> nRecs

Arguments

    rl is the RecordList to be counted.

Returns

    rlCount() returns the number of records in the given RecordList.

Description

    rlCount() counts the number of records in the given RecordList.

Example

    rl := rlNew(50)
    ? rlCount(rl)           && Will return 0 (initially no records)
    rlSet(rl, 20)           && Add a record
    rlSet(rl, 30)           && Add a record
    ? rlCount(rl)           && Will now return 2
    rlDestroy(rl)
