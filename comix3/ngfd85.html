Manual Organization
------------------------------------------------------------------------------

This chapter provides a brief overview of what each of the rest of the
chapters of this manual cover.

The Overview chapter provides a conceptual framework for understanding
what ClipMore does and how it works.

The Advanced Overview chapter describes the more sophisticated functions
which are provided by ClipMore for special requirements.

The Installation chapter explains how to install ClipMore.

The Quick Start chapter provides a brief description of how to best get
started using ClipMore.

The Optimizable Expressions chapter explains what indexes ClipMore will
use to optimize various queries.

The Command Reference chapter describes each ClipMore command and/or
commands which ClipMore enhances and provides a brief example of each
command's usage.

The Function Reference chapter describes each ClipMore function and
provides a brief example of each function's usage.

The Advanced Topics chapter covers a series of issues such as memory
requirements, use of overlay linkers, known limitations, etc.

The Appendixes cover error messages, technical support, and the license
agreement.

One brief point about our coding style.  We use upper case letters sparingly
in our code examples.  While this is considered poor style in some xBASE
circles, we do not feel compelled to SHOUT AT OUR COMPUTERS LIKE THIS TO
MAKE THEM DO SOMETHING.  We prefer to talk to them like this.
