rlMaybeEval()         Remove Maybe's from a RecordList
------------------------------------------------------------------------------

Synopsis

    rlMaybeEval(rl, bBlock) -> nRecs

Arguments

    rl is the RecordList from which Maybe's are to be removed.

    bBlock is the CodeBlock to be evaluated for each Maybe record.  It
    should return a logical (.T. if the record meets the condition, .F.
    otherwise).

Returns

    rlMaybeEval() returns the number of Maybe records visited.

Description

    rlMaybeEval() removes Maybe's from a RecordList (makes them either Yes
    or No).

    rlNewQuery() will return a RecordList containing Maybe's if the
    condition is not fully optimizable (see the discussion under
    cmDoLinear()).

    rlMaybeEval() will evaluate bBlock for all records which are Maybe's in
    rl.  This allows you to evaluate the condition for each record, and
    determine if it meets the condition or not.

    Note that if the RecordList has no Maybe's, then this function does
    nothing.

    The source code for cmDoLinear() gives you a good example of the use of
    this function.

Example

    *   Use demo (only index is age)
    use demo index age

    *   No index on salary, so can't optimize that"
    rl := rlNewQuery("age < 25 .and. salary <= 25000")
    ? "Approximate count: ", rlCount(rl)

    *   Clear off the Maybe's
    rlMaybeEval(rl, {|| age < 25 .and. salary <= 25000})

    *   Now give the exact count
    ? "      Exact count: ", rlCount(rl)
