rlOr()                OR together two RecordList's
------------------------------------------------------------------------------

Synopsis

    rlOr(rl1, rl2) -> rlResult

Arguments

    rl1 & rl2 are the two RecordList's to be OR'd together.

Returns

    rl1 (OR'd with rl2).

Description

    rlOr() OR's together two RecordList's so that records contained in
    either or both RecordList's are contained in the result.

    rl2 is destroyed by this function.  The returned RecordList is actually
    rl1 (with its bit-maps properly OR'd).  This saves the the developer
    from having to explicitly destroy the OR'd recordList.

Example

    rl1 := rlNew(10)
    rlSet(rl1, 5)
    rl2 := rlNew(10)
    rlSet(rl2, 7)
    ? rlCount(rl1), " records are in the first RecordList"
    ? rlCount(rl2), " records are in the second RecordList"
    rlResult = rlOr(rl1, rl2)
    ? rlCount(rlResult), " records are in either or both RecordList's"

    ? (rlResult == rl1)             && Will return .T.
    ? (rlResult == rl2)             && Will return .F.
    rlDestroy(rlResult)

See Also

    rlOr(), rlNot()
