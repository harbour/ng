rlPrevRecno()         Get the previous record in a RecordList
------------------------------------------------------------------------------

Synopsis

    rlPrevRecno(rl, nRecno) -> nPrevRecno

Arguments

    rl is the RecordList to be scanned.

    nRecno is the record number to start looking at (exclusive).  I.e., the
    scan will start at nRecno - 1.

Returns

    rlPrevRecno() returns the previous record number in rl.

    rlPrevRecno() will return 0 when there are no more record numbers in
    rl.

Description

    rlPrevRecno() allows you to get the record number before nRecno which is
    contained in the recordList.

Example

    rl := rlNew(100)
    rlSet(rl, 25)
    rlSet(rl, 75)
    rlFastOutput(rl)
    rlSlowOutput(rl)
    return

    *   This quickly outputs all the records contained in rl (backwards)
    function rlFastOutput(rl)
    local curRec := rlLen(rl) + 1
    do while ((curRec := rlPrevRecno(rl, curRec)) != 0)
        ? curRec
    enddo
    return

    *   This does the same thing, but is slower (since you have to go through
    *   one iteration of the loop for the entire rlLen())
    function rlSlowOutput(rl)
    local curRec
    for curRec := rlLen(rl) to 1 step -1
        if (rlTest(rl, curRec))
            ? curRec
        endif
    next curRec
    return
