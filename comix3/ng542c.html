cmKeySkip()           Skip the specified number of keys (filtered)
------------------------------------------------------------------------------

Synopsis

    cmKeySkip(nKeys) -> NIL

Arguments

    nKeys is the number of keys to skip (if negative, the skip will be
    backwards).

Returns

    NIL.

Description

    cmKeySkip() allows you to skip the specified number of filtered keys.

    cmKeySkip() obeys the current index scope (if any).  cmKeySkip() will
    move to BOF() if you attempt to skip past the beginning of the scope, or
    EOF() if you attempt to skip past the end of the scope.

Example

    index on age to age
    index on last to last
    use demo index age, last

    cmFilter("last = 'J'")
    go top
    ? cmKeyNo()                     && Will return 1
    cmKeySkip(5)                    && Skip forward 5 keys
    ? cmKeyNo()                     && Will return 6
    cmKeySkip(-3)                   && Skip backward 3 keys
    ? cmKeyNo()                     && Will return 3
