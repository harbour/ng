rlNew()               Create a RecordList
------------------------------------------------------------------------------

Synopsis

    rlNew([nLen]) -> rlResult

Arguments

    nLen is the length of the RecordList to be created.  If omitted, then
    the recCount() of the current database will be used.

Returns

    The RecordList (actually represented by a long).

Description

    rlNew() allows you to create a new RecordList which initially contains
    no records (i.e., has no bits set).

    When you are done with the RecordList, it must be destroyed.

Example

    rl := rlNew(50)                 && Create a new RecordList
    ? rlLen(rl)                     && Will return 50
    ? rlCount(rl)                   && Will return 0 (no recs in it)
    rlSet(rl, 10)                   && Add record 10
    rlSet(rl, 30)                   && Add record 30
    rlSet(rl, 70)                   && Add record 70 (won't do anything
                                    && because 70 is greater than the
                                    && length of the RecordList (50))
    ? rlCount(rl)                   && Will return 2 (10 & 30 are in it)
    rlDestroy(rl)                   && Get rid of it

Example

    use demo
    rl := rlNew()                   && Set up a new RecordList (default len)
    ? rlLen(rl)                     && Will return # of recs n database
    ? recCount()                    && Will return same as above
    rlDestroy(rl)                   && Get rid of it

See Also

    rlNewDup(), rlNewQuery()
