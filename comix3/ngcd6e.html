rlNot()               NOT a RecordList
------------------------------------------------------------------------------

Synopsis

    rlNot(rl) -> rlResult

Arguments

    rl is the RecordList to be NOT'd.

Returns

    rlNot() returns the resultant RecordList.  (This is the same as rl with
    the bit-maps of the RecordList appropriately NOT'd).

Description

    rlNot() NOT's the RecordList.  All Yes's in the RecordList are turned
    into No's, and all No's in the RecordList are turned into Yes's.

    Please note that if your RecordList has Maybe's (e.g., a RecordList
    returned by rlNewQuery() without a fully optimizable condition), Maybe's
    stay as Maybe's.

Example

    rl := rlNew(100)
    ? rlCount(rl)                   && Returns 0
    rl := rlNot(rl)
    ? rlCount(rl)                   && Returns 100

See Also

    rlAnd(), rlOr()
