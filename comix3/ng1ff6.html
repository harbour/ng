cmClrFilter()         Clear index filter previously set up by cmFilter()
------------------------------------------------------------------------------

Synopsis

    cmClrFilter() -> NIL

Arguments

    None.

Returns

    NIL.

Description

    The cmClrFilter() function reverses the effects of the cmFilter()
    function.  It is used when you have finished the processing of the
    records which were isolated by  cmFilter().

    It is equivalent to SET FILTER TO (with no filter condition).

Example

    use demo index age, lastcall

    *   Restrict to just these people
    cmFilter("age <= 30 .and. lastcall >= ctod('06/01/89')")

    list age, lastcall  && List 'em out
    ? dbFilter()        && "age >= 30 .and. lastcall ..."
    cmClrFilter()       && Clean up
    ? dbFilter()        && "" (filter has been cleared)
