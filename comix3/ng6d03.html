cmSetLinear()         Set whether or not to use Linear Optimization
------------------------------------------------------------------------------

Synopsis

    cmSetLinear(lOn) -> lPrevLinear

Arguments

    lOn is a logical determining whether or not Linear Optimization should
    automatically be used by cmFilter() and cmReFilter().  If lOn is .T.,
    then Linear Optimization will be used;  if .F., then Linear Optimization
    will not be used.

Returns

    cmSetLinear() the previous setting of Linear Optimization.

Description

    cmSetLinear() allows you to turn Linear Optimization on or off.  By
    default, it is off.

    See the discussion under cmDoLinear() for more information on
    Linear Optimization.

Example

    ? cmGetLinear()         && Returns .f. (default)
    cmSetLinear(.t.)        && Turn Linear Optimization on.
    ? cmGetLinear()         && Now returns .t.

See Also

    cmDoLinear(), cmGetLinear()
