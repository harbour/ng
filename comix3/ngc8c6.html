rlNextRecno()         Get the next record in a RecordList
------------------------------------------------------------------------------

Synopsis

    rlNextRecno(rl, nRecno) -> nNextRecno

Arguments

    rl is the RecordList to be scanned.

    nRecno is the record number to start looking at (exclusive).  I.e., the
    scan will start at nRecno + 1.

Returns

    rlNextRecno() returns the next record number in rl.

    rlNextRecno() will return 0 when there are no more record numbers.

Description

    rlNextRecno() allows you to get the record numbers of all records which
    are contained in the recordList.

Example

    *   This quickly outputs all the records contained in rl
    function rlFastOutput(rl)
    local curRec := 0
    do while ((curRec := rlNextRecno(rl, curRec)) != 0)
        ? curRec
    enddo
    return

    *   This does the same thing, but is slower (since you have to go through
    *   one iteration of the loop for the entire rlLen())
    function rlSlowOutput(rl)
    local curRec := 1
    local numRecs := rlLen(rl)
    for curRec := 1 to numRecs
        if (rlTest(rl, curRec))
            ? curRec
        endif
    next curRec
    return
