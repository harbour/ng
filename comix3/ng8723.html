rlDestroy()           Destroy (free memory for) a RecordList
------------------------------------------------------------------------------

Synopsis

    rlDestroy(rl) -> NIL

Arguments

    rl is the RecordList to be destroyed.

Returns

    NIL.

Description

    rlDestroy() destroys the memory allocated for a RecordList.

    Anytime you create a new RecordList (using one of the rlNew*()
    functions), you must eventually destroy it.

    rlDestroy() allows you to discard a RecordList once you are done
    using it.

Example

    rl1 := rlNew()          && Create a new RecordList
    rl2 := rlNew()          && Create another RecordList
    rl1 := rlAnd(rl1, rl2)  && AND them together (this will automatically
                            && destroy rl2)
    rlDestroy(rl1)          && Now destroy rl1 (we're all cleaned up)
