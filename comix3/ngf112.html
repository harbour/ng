rlTest()              Determine if a record is in a RecordList
------------------------------------------------------------------------------

Synopsis

    rlTest(rl, nRecno) -> lPresent

Arguments

    rl is the RecordList to check.

    nRecno is the record number to check for.

Returns

    rlTest() returns a logical.  It will return .T. if nRecno is in rl.  It
    will return .F. otherwise.

Description

    rlTest() allows you to determine if a record is contained in a
    RecordList.

Example

    rl := rlNew(50)
    ? rlTest(rl, 25)            && Will return .f.
    rlSet(rl, 25)               && Add record 25
    ? rlTest(rl, 25)            && Will return .t.
    rlClear(rl,25)              && Remove record 25
    ? rlTest(rl, 25)            && Will return .f. (no longer in RecordList)
    rlDestroy(rl)               && Get rid of the RecordList

See Also

    rlClear(), rlSet()
