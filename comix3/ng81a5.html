rlClear()             Remove a record from (clear a bit in) a RecordList
------------------------------------------------------------------------------

Synopsis

    rlClear(rl, nRecno) -> NIL

Arguments

    rl is the RecordList from which nRecno is to be removed.

    nRecno is the record number to be removed from the RecordList.

Returns

    NIL.

Description

    rlClear() allows you to remove records from a RecordList.

Example

    rl := rlNot(rlNew(100)) && Get RecordList with all records Yes
    ? rlCount(rl)           && Will return 100
    rlClear(rl, 50)
    rlClear(rl, 75)
    ? rlCount(rl)           && Will return 98
    rlDestroy(rl)           && Get rid of the RecordList

See Also

    rlSet(), rlTest()
