rlGetFilter()         Get the Filter RecordList
------------------------------------------------------------------------------

Synopsis

    rlGetFilter() -> rl

Arguments

    None.

Returns

    rlGetFilter() returns the Filter RecordList for the current workarea.

Description

    rlGetFilter() returns the Filter RecordList for the current workarea,
    _but_ ClipMore retains ownership of it.

    This allows you to quickly manipulate the Filter RecordList as needed.

Example

    *   Restrict to unpaid invoices over 30 days old
    cmFilter("invDate <= (date() - 30) .and. .not. paid")

    *   Let user browse them (to clear off people he doesn't want to bill)
    browseIt()

    *   Now do the past due invoicing
    ...

    *   browseIt() is too complicated to show here.  The idea is that you'll
    *   have a key which when pressed will cause a call to removeFromList()

    *   Discard the current record from the RecordList
    function removeFromList()
    rlClear(rlGetFilter(), recno())
    return
