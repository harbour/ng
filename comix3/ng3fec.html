cmGetLinear()         Get the current setting of Linear Optimization
------------------------------------------------------------------------------

Synopsis

    cmGetLinear() -> lLinearUsed

Arguments

    None.

Returns

    cmGetLinear() returns a logical which is .T. if Linear Optimization will
    automatically be used by cmFilter(), and .F. if Linear Optimization will
    not be used.

Description

    cmGetLinear() returns whether or not Linear Optimization is used for
    cmFilter() and cmReFilter().

    Linear Optimization causes any Maybe's due to a partially optimizable
    condition to be removed.

Example

    ? cmGetLinear()         && Returns .f. (default)
    cmSetLinear(.t.)        && Turn Linear Optimization on.
    ? cmGetLinear()         && Now returns .t.

See Also

    cmDoLinear(), cmSetLinear()
