ClipMore isn't improving the speed of my filters!  Why?

ClipMore/CMX will only work with Comix, _not_ DBFNTX.  You must link with
both Comix & ClipMore.

If you are using the SET FILTER TO command, you must #include ClipMore.ch in
that PRG to optimize the command.
