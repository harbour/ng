cm2Str()              Convert variable to equivalent string constant
------------------------------------------------------------------------------

Synopsis

    cm2Str(xValue) -> cValue

Arguments

    xValue is the value to be converted to its string constant equivalent.

Returns

    cm2Str() returns the string constant equivalent of the given value.

Description

    The cm2Str() function allows you to straightforwardly use local or
    static variables within a query expression.

    Local and static variables cannot be referenced directly because there
    is no way for ClipMore to associate the symbol name with its value.
    However, cm2Str() provides a straightforward means of working around
    this limitation by converting the value to an equivalent string constant
    which can be concatenated together as part of the query string.

Example

    function test()
        local cStr := "abc"
        local dDate := ctod("04/22/94")
        local lLogical := .f.
        local nValue := 13.7

        ? cm2Str(cStr)          && "abc"
        ? cm2Str(dDate)         && ctod('04/22/94')
        ? cm2Str(lLogical)      && .f.
        ? cm2Str(nValue)        && 13.7

        use demo index age, lastcall

        cmFilter("age >= " + cm2Str(nValue))
        list age

        cmFilter("lastcall < " + cm2Str(dDate))
        list lastcall
    return nil
