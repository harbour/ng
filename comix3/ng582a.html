cmReFilter()          Rapidly isolate records meeting additive condition
------------------------------------------------------------------------------

Synopsis

    cmReFilter(cCondition) -> NIL

Arguments

    cCondition is the additional condition which records should meet.  This
    condition is AND'd with the current filter set up by a previous
    invocation of cmFilter() or cmReFilter().

Returns

    NIL.

Description

    The cmReFilter() function is very similar to cmFilter(), but it further
    filters previously selected records.  The cCondition is simply AND'd
    with the previously existing filter (if any).

    The advantage of using this function is that none of the work done in
    previously selecting the records is repeated.

Example

    use demo index last, age, lastcall
    * Get records;  no previous filter, but works AOK
    retVal = cmReFilter("age < 25")
    ? retVal, "records meet condition: " + dbfilter()
    * Now further select on lastcall
    retVal = cmReFilter("lastcall <= ctod('01/01/75')")
    ? retVal, "records meet condition: " + dbfilter()
    * Now further select on street
    retVal = cmReFilter("street = '1'")
    ? retVal, "records meet condition: " + dbfilter()
    * Bring database out of use
    cmClose()
