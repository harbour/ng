 Symbol Class - Instance Variable
 name
------------------------------------------------------------------------------

    This variable is read-only, and contains the name of the symbol
    referred to by the object.  Note that this string is not actually
    stored in the object - it is obtained from the Clipper symbol table
    when the name message is sent.  This ensures that Symbol objects
    are small and efficient.
