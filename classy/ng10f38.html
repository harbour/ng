  Viewing class variables

  An instance variable called class appears (in lower case) at the end
  of the normal list of instance variables. Selecting this instance variable
  will display a list of the class variables for that object's class. To see
  an example of this, try inspecting the OBOX variable in the example
  program HERITEST.
