  Standard Object Inspector

  If the Class(y) 2.4 Enhanced Object Inspector, described in the previous
  section, is not linked into an application, a slightly enhanced version of
  the standard Clipper object inspector is used.

  To support the expanded capabilities Class(y) provides over Clipper, three
  enhancements have been made to the way in which objects appear in the
  debugger's object inspector.
