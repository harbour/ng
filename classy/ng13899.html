  Overlaying

  All modules in CLASSY.LIB can be overlayed with the exception of
  _CYCLASS.OBJ and CSYXSEND.OBJ, which must be linked into the root of an
  application.

  The enhanced object inspector contained in CSYINSP.LIB can be completely
  overlayed. It is implemented in Clipper, so it will usually be overlayed
  automatically depending on which linker and linker mode is being used.
