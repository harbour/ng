 VISIBLE:

 Synopsis

         Cause subsequent messages to be exported

 Syntax

         VISIBLE:

 Description

         This command is equivalent to the EXPORT: command. It is
         provided for symmetry with the HIDDEN: command. Its use is
         optional.
