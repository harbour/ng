 CLASS MESSAGE...[IS...] TO

 Synopsis

         Forward class message to class variable

 Description

         The behavior of this command is identical to that of the
         MESSAGE...[IS...] TO command, except that it defines a class
         message which is forwarded to a class variable, instead of an
         ordinary message which is forwarded to an ordinary variable.

 See Also
