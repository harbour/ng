 EXPORT:

 Synopsis

         Cause subsequent messages to be exported

 Syntax

         EXPORT:

 Description

         This command causes subsequent variable, message and method
         declarations in a class specification to be exported, ie. made
         accessible to methods and functions outside of the class.
