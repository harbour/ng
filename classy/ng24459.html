 Symbol Class - Instance Variable
 nSym
------------------------------------------------------------------------------

    This variable is declared as HIDDEN: and cannot ordinarily be assigned
    to or accessed, although it is visible in the debugger.  It contains a
    number which corresponds internally to the symbol referred to.
