 SET RMAKE
 Set default RMAKE command line options
-------------------------------------------------------------------------------

    SET RMAKE=<option list>

 Arguments

    <option list> is a list of command line options to use each
    time RMAKE is executed.

 Description

    The RMAKE environment variable specifies a list of command line options
    processed each time RMAKE.EXE is executed.  Options defined in the
    environment variable are overridden by the same options specified on
    the RMAKE command line.

 Examples

    SET RMAKE=/B /W
