 _mCol()
 Retrieve the column position of the mouse pointer
------------------------------------------------------------------------------
 C Prototype

     #include "llibg.api"
     int _mCol(void)

 Returns

     The mouse's column position as a text-based coordinate.

 Files   Library is LLIBG.LIB, header file is Llibg.api.
