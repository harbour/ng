 _mPixY()
 Retrieve the pixel-based Y coordinate of the mouse pointer
------------------------------------------------------------------------------
 C Prototype

     #include "llibg.api"
     int _mPixY(void)

 Returns

     The mouse's pixel-based Y coordinate position.

 Files   Library is LLIBG.LIB, header file is Llibg.api.
