 _gFntErase()
 Erase a font previously loaded with _gFntLoad() from memory
------------------------------------------------------------------------------
 C Prototype

     #include "llibg.api"
     void _gFntErase(
                      int iFontID
                    )

 Arguments

     iFontID is the font handle as returned by _gFntLoad().

 Files   Library is LLIBG.LIB, header file is Llibg.api.
