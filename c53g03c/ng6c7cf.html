 LLG_POINT structure
 Structure for specifying coordinates of a point
------------------------------------------------------------------------------
 Structure

     typedef struct
     {
        int   iX;
        int   iY;
     } LLG_POINT;
     
     typedef LLG_POINT far * LLG_LPPOINT;

 Elements

     iX is the pixel-based X coordinate.

     iY is the pixel-based Y coordinate.

 Files   Header file is Llibg.api.

 Used By   _gPolygon()
