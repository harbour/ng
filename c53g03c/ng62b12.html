 _mPixX()
 Retrieve the pixel-based X coordinate of the mouse pointer
------------------------------------------------------------------------------
 C Prototype

     #include "llibg.api"
     int _mPixX(void)

 Returns

     The mouse's pixel-based X coordinate position.

 Files   Library is LLIBG.LIB, header file is Llibg.api.
