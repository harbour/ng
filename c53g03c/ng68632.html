 _gModeSet()
 Switch the video mode
------------------------------------------------------------------------------
 C Prototype

     #include "llibg.api"
     ERRCODE _gModeSet(
                        int iMode
                      )

 Arguments

     iMode is the video mode (see _gModeGet() for a table of possible
     values).

 Returns

     SUCCESS if successful; otherwise, an error code.

 Files   Library is LLIBG.LIB, header file is Llibg.api.
See Also: _gModeGet()
