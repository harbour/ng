 _mRow()
 Retrieve the row position of the mouse pointer
------------------------------------------------------------------------------
 C Prototype

     #include "llibg.api"
     int _mRow(void)

 Returns

     The mouse's row position as a text-based coordinate.

 Files   Library is LLIBG.LIB, header file is Llibg.api.
