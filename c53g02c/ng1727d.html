 4057:  Premature end of file encountered, file ignored (Serious)

     Explanation:  CA-Clipper/Exospace encountered the end of an object
     file before it encountered the MODEND record, which marks the logical
     end of an object file.  CA-Clipper/Exospace ignores the invalid object
     file.
