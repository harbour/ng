 R3003   Invalid option: '<option>'

     Explanation:  An invalid option was specified on the command line or
     in the RMAKE environment variable.  The available options can be found
     in the "Program Maintenance--RMAKE.EXE" chapter in the Programming and
     Utilities Guide.

     Action:  Correct and run the make again.
