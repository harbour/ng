 4034:  Not enough file handles to perform link (Fatal) 

     Explanation:  CA-Clipper/Exospace reuses file handles after it reads
     a file into memory, starting with the least recently used, but you must
     have enough handles for all open output files, and three extra.

     Action:  Make sure that your CONFIG.SYS file specifies enough file
     handles.
