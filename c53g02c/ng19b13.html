 1002:   duplicate input file <filename> ignored

     Explanation:  This message indicates that the same input file
     appeared in more than one FILE or LIBRARY command.  The duplicated input
     file will be ignored.

     See also:  The Linker Reference Commands section in the "CA-Clipper
     Real Mode Linker--BLINKER.EXE" chapter of the Programming and Utilities
     Guide.
