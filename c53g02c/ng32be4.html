 DBFNDX/1010  Read error

     Explanation:  A read error occurred on the specified file.

     Action:  Some suggestions to resolve the problem are:

     1. If the file is on a floppy disk, make sure the disk is properly
        seated in the drive.

     2. In a network environment, make sure the network connection is still
        valid; check for problems at the server.

     See Also:  "Network Programming" chapter in the Programming and
     Utilities Guide
