 1001:   ignoring invalid link script file command parameter in '<command>'

     Explanation:  This message indicates that the link script file
     command specified has been ignored as it contains an invalid parameter
     value.

     See also:  The Linker Reference Commands section in the "CA-Clipper
     Real Mode Linker--BLINKER.EXE" chapter of the Programming and Utilities
     Guide.
