 DBG0003 Unable to execute Windows/DOS communications client program.
         Program not found.

     Explanation:  CACI.BAT could not be found.

     Action:  Be sure that the directory which contains CACI.BAT
     (normally C:\CLIP53\BIN) is included in the current PATH.
