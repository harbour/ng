 1108:   stack overflow during linking

     Explanation:  This message indicates that the runtime stack for
     BLINKER.EXE has overflowed and should not be displayed under normal
     circumstances.

     Action:  If this message is displayed, reinstall the Blinker
     software.  if the error persists, contact Computer Associates technical
     support.
