 C2023   CA-Clipper definition of CALLed symbol

     Explanation:  A previously CALLed procedure name has been declared
     in a PROCEDURE or FUNCTION statement.

     Action:  This conflict can only be resolved by renaming either the
     CALLed procedure name or the declared procedure or function name.
