 335     String/Array invalid pointer

     Explanation:  This results from an error accessing the memory
     address for a string value during an element assignment while declaring
     a large multidimensional array.

     Action:  Try declaring an empty array and then build the array using
     AADD().  Check string assignments for a NULL character.
