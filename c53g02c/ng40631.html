 5320    VMM IAMBS Manager Error

     Explanation:  An "out of memory" error has occurred because the VM
     system is unable to provide more memory, or it encountered an error.

     Action:  Make more conventional memory available to the VM system
     and test with a clean environment after a cold boot.

     See Also:  Error 5302, "The Runtime Environment" chapter of the
     Programming and Utilities Guide.
