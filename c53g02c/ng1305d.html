 1:  "Insufficient memory (extended)"

     Explanation:  There is not enough extended memory to load your
     program.

     Action:  check the memory to make sure that memory above 1MB is
     configured as extended (not expanded) memory, or use a VCPI host (such
     as QEMM or EMM386).
