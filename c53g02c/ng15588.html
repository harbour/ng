 4008:  Can't find indicated file (Serious) 

     Explanation:  You have specified a file without adequate path
     information, or the file does not exist.  For .OBJ files,
     CA-Clipper/Exospace searches the current directory first, then looks in
     the path set in the DOS environment variable OBJ.
