 C2043   Invalid alias expression

     Explanation:  You specified an identifier on the left side of the
     alias operator (->) that is not a valid alias name.  The left side of
     the alias operator can only be an identifier or a macro expression.

     Action:  Check to see that you have not specified an expression as
     the alias name.
