 DBFCDX/1050  Invalid TAG/ORDER

     Explanation:  You tried to set the tag order of the order bag to a
     non-existing tag or an illegal (blank) tag name.

     Action:  Make sure that OrdSetFocus() is setting the tag order to a
     valid  tag.
