 DBCMD/1011   Alias already in use

     Explanation:  An attempt was made to create an alias that is already
     in use in another work area.  The error is caused by:

     USE Test NEW

     USE Test NEW  // error occurs here

     Action:  If created due to the default alias, supply a valid
     identifier with the ALIAS clause.
