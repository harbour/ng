 4046:  Unsupported fixup type found (Warning) 

     Explanation:  CA-Clipper/Exospace has found a fixup type that it
     does not recognize.  CA-Clipper/Exospace recognizes all Microsoft fixup
     types, but the Intel OMF specification defines additional fixup types
     that CA-Clipper/Exospace does not recognize.
