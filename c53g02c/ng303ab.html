 DBFCDX/631   Non-compact index not supported

     Explanation:  The .cdx file was found to be a NON-COMPACT .cdx file.

     Action:  Check to see if the file was created under FOXPRO v1.0.
     CDX RDD only supports COMPACT version of .cdx files.
