 BASE/1068    Argument error: array access 

     Explanation:  When attempting to retrieve the value of an array
     element, the subscript value was non-numeric, or the variable being
     subscripted was not an array.

     Action:  Correct the program.

     See Also:  "Basic Concepts" chapter in the Programming and Utilities
     Guide
