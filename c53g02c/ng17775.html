 4067:  Out of memory. Last module processed = <modulename>. -(Serious)

     Explanation:  Any modules linked after this module will not have
     symbols.  Symbols for modules are processed based on the order specified
     to the linker.

     Action:  Try compiling object modules without debugging information.
