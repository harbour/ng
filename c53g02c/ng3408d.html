 DBFNDX/1027  Limit exceeded

     Explanation:  Too many indexes were opened for a work area.

     Action:  Reduce the number of active indexes for the work area.  For
     each work area, a maximum of 15 indexes can be active at any one time.

     See Also:  USE command, SET INDEX command
