 773     Dynamic overlay system not linked in

     Explanation:  This may indicate an invalid link line or a corrupted
     executable file.

     Action:  Make sure the latest libraries are used, and that no
     third-party libraries are linked.  Determine that the link instructions
     are valid and the .EXE is not corrupted.  Confirm that the memory
     configuration did not affect linking.
