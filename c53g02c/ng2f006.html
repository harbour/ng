 DBCMD/1014   Invalid argument for DBCREATE()

     Explanation:  This error occurs when the array for DBCREATE() has an
     invalid array element.

     Action:  Some suggestions to resolve the problem are:

     1. Make sure that all array elements for the DBCREATE() function contain
        valid data.

     2. Check for an additional comma after the fourth array element, as that
        will cause this error.
