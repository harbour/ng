 1134:   error reading file <filename>

     Explanation:  Blinker was unable to read from the specified file.

     Action:  Check to make sure the file is accessible via the
     appropriate PATH and other environment variables, or is specified with a
     full path in the link script.  Alternatively, the file may be in use or
     corrupt.
