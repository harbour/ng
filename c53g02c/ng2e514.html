 DBCMD/1004   Argument error: SET RELATION TO or DBSETRELAT

     Explanation:  One or more of the arguments to SET RELATION TO or
     DBSETRELAT() was of the wrong type, or the alias was invalid.

     Action:  Correct the program.

     See Also:  DBSETRELAT() function, SET RELATION TO command
