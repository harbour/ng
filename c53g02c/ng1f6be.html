 1147:   <filename(modulename)> contains incremental compilation errors

     Explanation:  This message indicates that the specified .OBJ file
     was created with a compiler which supports incremental compilation, but
     that the compilation failed.

     Action:  Correct and recompile the specified .OBJ file and then
     relink.
