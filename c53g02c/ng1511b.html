 4003:  Unable to process Microsoft Link command stream (Fatal)

     Explanation:  CA-Clipper/Exospace cannot interpret the Microsoft
     Link commands or script file.

     Action:  Make sure you used valid Microsoft syntax in the command
     line or script file.
