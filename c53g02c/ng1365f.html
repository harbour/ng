 8:  "Cannot open file %s" (filename)

     Explanation:  The DOS/16M loader cannot open the executable file
     (file name).  If you receive this message after executing the command
     loader file name, the file name is probably incorrect.  Otherwise, DOS
     may not have enough file units configured (the FILES= entry in
     CONFIG.SYS should be larger).
