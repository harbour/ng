 4087:  Illegal index field in fixup (Serious)

     Explanation:  CA-Clipper/Exospace encountered a malformed or
     unsupported fixup in the object file.  If you use a Borland compiler
     without the Vs switch, you will get this error.

     Action:  Recompile the object file; if the problem persists, call
     Computer Associates Technical Support.
