 Preprocessor

     (program, preprocessor) A translation program that prepares source code
     for compilation by applying selective text replacements.  The
     replacements to be made are specified by directives in the source file.
     In CA-Clipper, the preprocessor operates transparently as a part of the
     compiler program (CLIPPER.EXE).
