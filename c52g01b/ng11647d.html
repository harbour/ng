 View

     (database) A DBMS term that defines a virtual table.  A virtual table
     does not actually exist but is derived from existing tables and
     maintained as a definition.  The definition in turn is maintained in a
     separate file or as an entry in a system dictionary file.  In
     CA-Clipper, views are supported only by DBU.EXE and are maintained in
     (.vew) files.
