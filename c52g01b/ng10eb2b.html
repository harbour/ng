 Positional

     (linker) The POSITIONAL command interface requires that certain items
     appear on the input line in a specific order.  This syntax is similar to
     Microsoft LINK interface.  Because this syntax limits the use of .RTLink
     overlays to one overlay area, it is recommended that the FREEFORMAT
     syntax be used.
