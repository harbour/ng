 Picture

     (user interface) A string that defines the format for data entry or
     display in a GET, SAY, or the return value of TRANSFORM().  Picture
     strings are comprised of functions which affect the formatting as a
     whole and a series of template characters that affect formatting on a
     character by character basis.
