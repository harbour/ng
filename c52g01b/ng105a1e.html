 Dumb Stringify Result Marker

     (preprocessor) A result marker of the form #<{id}>.  id must correspond
     to the name of a match marker.  A dumb stringify result marker specifies
     that the corresponding source text is to be enclosed in quotes.  If the
     matched source text constitutes a list of expressions, each expression
     in the list is individually stringified.  If no source text was matched,
     an empty pair of quotes is produced.
