 Menu

     (user interface) An onscreen list of choices from which the user
     selects.  Menus range from simple to elaborate forms.  Two examples are
     menus that pull down from the top of the screen (an elaborate type
     requiring more programming), or a simple list of numbered items from
     which the user selects by entering the appropriate number.
