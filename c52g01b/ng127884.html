     Pos

        Contains a numeric value indicating the position of the cursor within
        the editing buffer.  Get:Pos is meaningful only when the Get object
        has input focus.  At all other times, it contains NIL.

        oGet:SetFocus()
        If (oGet:Pos == 3)
           .
           .
           .

        Endif
