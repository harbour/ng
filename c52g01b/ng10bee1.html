 Memo Type

     (data type, database) A special database field type consisting of one or
     more characters in the IBM extended character set.  The maximum size of
     a memo field in CA-Clipper is 65,534 bytes.  A memo field differs only
     from a character string by the fact it is stored in a separate memo
     (.DBT file) and the field length is variable-length
