 Special Operators
-------------------------------------------------------------------------------
 Operator      | Description
               |
 :             | Send a message to an object
 ()            | Function or grouping
 []            | Array element
 {}            | Constant array definition
 ->            | Alias identifier
 &             | Compile and execute
 @             | Pass by reference
