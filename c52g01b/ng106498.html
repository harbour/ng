 Environment Variables

     (configuration) Operating system variables that can be used to
     communicate configuration information to executable programs.
     Environment variables are manipulated using the DOS SET command.  The
     CA-Clipper compiler and linker respond to certain environment variables.
     CA-Clipper programs can inspect the settings of environment variables
     using the GETENV() function.
