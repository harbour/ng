     ErrorNew() --> objError

        Returns a new Error object.

        oNewErr := ErrorNew()

        // Assign some instance variables
        oNewErr:CanDefault    := .F.
        oNewErr:CanSubstitute := .F.
        oNewErr:CanRetry      := .F.
        oNewErr:Severity      := ES_CATASTROPHIC
        oNewErr:Description   := ERR_UNKNOWN
        oNewErr:Operation     := ERR_WHOKNOWS
