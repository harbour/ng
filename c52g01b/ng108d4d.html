 Index

     (database) An ordered set of key values that provides a logical ordering
     of the records in an associated database file.  Each key in an index is
     associated with a particular record in the database file.  The records
     can be processed sequentially in key order, and any record can be
     located by performing a SEEK operation with the associated key value.
