 Normal Stringify Result Marker

     (preprocessor) A result marker of the form <"id">.  id must correspond
     to the name of a match marker.  A normal stringify result marker
     specifies that the corresponding source text is to be enclosed in
     quotes.  If the matched source text is a list, each element of the list
     is individually stringified.  If no source text was matched, an empty
     result is produced.
