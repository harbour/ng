 Script File

     (configuration) A text file that contains command input to a compiler,
     linker, or other utility program.  A script file is often used in lieu
     of equivalent keyboard input.  For the CA-Clipper compiler, script files
     contain a list of source files to be compiled into a single object file.

     (debugger) A file in which frequently used debugger commands are stored
     and from which those commands can be executed.
