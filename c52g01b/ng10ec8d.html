 Prelinked Library

     (linker) Part of the executable program that is stored external to the
     .EXE file.  Prelinked libraries are created before producing an
     executable in a multistep link.  This allows you to create a runtime
     library with code that you access from different programs, considerably
     speeding up the linking process.
