     Assign() --> self

        Assigns the value in the editing buffer to the Get variable by
        evaluating Get:Block with the buffer value supplied as its argument.
        This message is meaningful only when the Get object has input focus.

        oGet:SetFocus()
        If (oGet:Changed)
           oGet:Assign()

        Endif
