 Command

     (command) A statement to be translated by the CA-Clipper preprocessor
     into source code that will perform a particular operation.  All
     CA-Clipper commands are defined in the standard header file, STD.CH,
     located in \CLIPPER5\INCLUDE.  Also, the preprocessor directives that
     define a command.
See Also: Header File Statement STD.CH
