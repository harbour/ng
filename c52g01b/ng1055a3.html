 Directory

     (file) The major operating system facility for cataloging files.  A
     directory contains a list of files and references to child directories
     (subdirectories), and is identified by name.  Directories can be nested
     forming a hierarchical tree structure.  The operating system provides a
     number of facilities that allow users to create and delete directories.
