 Console Input/Output

     (user interface) A style of operation of the keyboard and display that
     emulates a simple typewriter-like interface.  Console input echoes each
     key typed and provides processing for the backspace and return keys.
     Console output wraps to the next line when the output reaches the right
     edge of the visible display, and scrolls the display when the output
     reaches the bottom of the visible display.
