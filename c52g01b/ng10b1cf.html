 Logify Result Marker

     (preprocessor) A result marker of the form #<.id.>.  id must correspond
     to the name of a match marker.  This result marker writes true (.T.) to
     the result text if any input text is matched; otherwise, it writes false
     (.F.) to the result text.  The input text itself is not written to the
     result text.
